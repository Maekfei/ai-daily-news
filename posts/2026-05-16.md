## 🎯 Agent 重点

### 1. Linux Foundation 接管 Agent Communication Protocol (ACP)
- **要点**：MCP 之外，Agent 间通信协议进入标准化轨道。
- **详情**：由 IBM、Anthropic、Cisco 联合发起的 ACP（Agent Communication Protocol）草案 v0.3 公开评议，目标统一不同框架 Agent 之间的消息格式与能力发现机制。
- **影响**：补全 Agent 互联生态，是继 MCP（工具）、A2A（任务委派）后第三块关键拼图。
- 🔗 [ACP draft v0.3 announcement](https://www.linuxfoundation.org/)

### 2. Devin 发布企业自治编程 Agent v3
- **要点**：宣称在 SWE-bench Verified 上达到 78.5%。
- **详情**：v3 新增长周期任务规划、审计日志与 Slack/Linear 双向集成，定价从按席位改为按完成 PR 计费。
- **影响**：编程 Agent 进入"按结果付费"时代，与 Cursor、Copilot Workspace 形成差异化竞争。
- 🔗 [Cognition Devin v3 Launch](https://www.cognition.dev/)
