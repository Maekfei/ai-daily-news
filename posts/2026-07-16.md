## 🎯 Agent 重点

### 1. xAI 正式开源 Grok Build，Rust 版终端编程代理入场
- **要点**：马斯克旗下 xAI 把此前仅向 $300/月 SuperGrok Heavy 订户封闭测试的编程代理 **Grok Build** 直接甩到 GitHub。
- **详情**：埃隆·马斯克旗下 **xAI** 于当地时间 7 月 15 日宣布将编程构建工具 **Grok Build** 源代码开源，同步上传至 GitHub。Grok Build 是一款 **Rust 语言编写**的终端 AI 编程代理（TUI），支持 headless CI 集成与 ACP（Agent-Client Protocol），核心亮点是把"同题多解 → git worktree 隔离 → best-of-n 择优"封装为默认工作流。该工具 5 月上市时曾因隐私争议登上安全新闻——The Hacker News 此前曝出 Grok Build CLI 默认将完整 Git 仓库（含历史与凭证）上传至 xAI 云存储。
- **影响**：叠加 OpenAI Codex Micro（今日发售）、Anthropic Claude Code 泄露源码事件、字节 DeerFlow 2.0（64k Star），2026 下半年"编程 Agent"赛道从闭源商业竞速转入开源基础设施战——xAI 用开源换开发者生态、争夺 Claude Code 与 Cursor 的既有份额。
- 🔗 [aitoolly：xAI Open Sources Grok Build (Rust TUI)](https://aitoolly.com/ai-news/article/2026-07-16-xai-open-sources-grok-build-a-powerful-rust-based-terminal-ai-coding-agent-for-developers)
- 🔗 [CNMO 中文：xAI 开源核心构建工具 Grok Build](https://ai.cnmo.com/news/813388.html)

### 2. OpenAI 首款硬件 Codex Micro 键盘今日发售，$230 定价
- **要点**：OpenAI 联合前苹果周边厂商 Work Louder 推出 **专为 Codex AI 编程代理设计的可编程宏键盘**，正式进入硬件市场。
- **详情**：OpenAI 于 2026 年 7 月 15 日正式发售旗下首款硬件产品 **Codex Micro**——一款与硬件厂商 Work Louder 合作的紧凑型机械键盘，含 **13 个机械键 + 摇杆 + 旋转编码器**，售价 **$230**，通过触觉反馈加速 Codex agent 的常用操作流程（run/plan/rollback/diff review 等）。这是继 OpenAI 与 Broadcom 联合发布首款自研 AI 推理芯片、以及与前苹果设计官 Jony Ive 合作"Gumdrop"消费级设备之后的第三条硬件线；发布当天 OpenAI 仍身陷苹果发起的商业机密诉讼中。同期，Codex 周活用户从 7 月 12 日的 600 万一路飙升到 7 月 15 日的 **800 万**。
- **影响**：编程 Agent 的战场从"云端 IDE 插件"外溢到物理终端；OpenAI 借此把开发者 Agent 从"随手用"变成"专机专用"，也为后续 Jony Ive 消费级硬件试水订阅硬件绑定的商业模式。

### 3. 阿里发布 Qwen-Audio-3.0-Realtime，Agent 工具调用能力对标 GPT-Realtime-2
- **要点**：**7 月 15 日**阿里云发布实时语音交互对话模型 Qwen-Audio-3.0-Realtime，同步推出 Plus / Flash 双版本。
- **详情**：Qwen-Audio-3.0-Realtime 在 **智商、Agent 工具调用、共情对话、双工交互流畅度**四条主线同步升级。其 **Preview 版**此前已在 Artificial Analysis 榜单夺得两项冠军、超越 OpenAI GPT-Realtime-2；正式版分为推理更强的 **Plus** 与速度更快的 **Flash** 两档，保持毫秒级时延响应，主打路线规划、智能客服、教育培训与情感陪伴场景——用户无需说出"帮我查一下"，模型就能主动推断意图并调用工具。
- **影响**：语音 Agent 从"能听会说"（Speech-to-Speech LLM）真正跨到"能听会做"（Speech-Agent）；结合 Qwen 端侧被苹果选中（见下一条），阿里把国内语音入口、手机端侧和企业客服 Agent 一次性接入。
- 🔗 [雷锋网：又快又聪明，阿里发布 Qwen-Audio-3.0-Realtime](https://www.leiphone.com/category/industrynews/Yg2RqalAz09LS6WJ.html)
