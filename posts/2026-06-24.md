## 🎯 Agent 重点

### 1. 智谱港股市值首破万亿港元，GLM-5.2 跻身全球第一梯队
- **要点**：上市不到半年累计涨超 1900%，国产大模型首获万亿级资本定价。
- **详情**：6 月 22 日，"全球大模型第一股"智谱（HK 02513）盘中涨幅一度超过 42%，最高触及 2980 港元/股，对应总市值 1.27 万亿港元；收盘报 2410 港元/股，市值 1.07 万亿港元。这一估值跃升发生在 6 月 17 日发布并开源 GLM-5.2 之后——新模型主攻"长程任务"，提供 1M 长上下文，能像人一样连续工作数小时自主完成完整工程，在 Artificial Analysis 综合榜单上跻身全球第一梯队，API 价格约为同级闭源模型的 1/5~1/8，并 Day 0 适配华为昇腾、寒武纪、昆仑芯等 9+ 国产算力平台。
- **影响**：稀缺性 + 长程 Agent 能力让一级、二级市场同时给出溢价；"模型即资产"叙事在国内进入实质兑现期。

- 🔗 [新浪财经：智谱市值破万亿港元 国产大模型获稀缺性溢价](https://finance.sina.com.cn/roll/2026-06-23/doc-iniekknt9747063.shtml)

### 2. OpenAI 安全专用 GPT-5.5-Cyber 完整版上线，CyberGym 得分 85.6%
- **要点**：在网络安全防御场景刷新 SOTA，比 GPT-5.5 提升 3.8 个百分点。
- **详情**：6 月 23 日，OpenAI 通过受限发布机制，向可信防守方推出 GPT-5.5-Cyber 的完整版本（此前仅开放预览版）。该模型在 CyberGym 攻防仿真基准上取得 85.6% 的成绩，高于 GPT-5.5 基础版的 81.8%，具备自主完成漏洞复现、攻击链分析、补丁建议等长链路任务的能力。模型仅向通过身份审查的安全研究机构、CERT 与企业蓝队开放，OpenAI 同步公布了滥用防护与日志审计要求。
- **影响**：标志着 Agent 模型从"通用助手"向"垂直领域专家"分化，安全行业开始进入"Cyber-Agent vs Cyber-Agent"对抗时代。
- 🔗 [新浪财经：战胜 Mythos 5，OpenAI 安全专用 GPT-5.5-Cyber 完全体来了](https://finance.sina.com.cn/tech/roll/2026-06-23/doc-iniekknt9717166.shtml)

### 3. SpaceX 600 亿美元全股票收购 AI 编程独角兽 Cursor
- **要点**：马斯克补齐 AI 编程工具拼图，对标 Anthropic / OpenAI 的 Coding Agent 战线。
- **详情**：6 月 16 日，SpaceX 与 Anysphere（Cursor 母公司）签署最终合并协议，按 Cursor 约 600 亿美元的隐含股权估值进行换股，全资子公司 X67 将并入 Cursor，合并后 Cursor 成为 SpaceX 全资子公司。交易不涉及现金，预计 2026 年 Q3 完成，仍需监管审批。距 SpaceX 纳斯达克 IPO 仅过去数日，距双方宣布合作意向不到两个月。
- **影响**：Coding Agent 赛道首个百亿美元级 M&A，行业从"框架并发"进入"巨头并购整合"阶段；xAI / Tesla / SpaceX 拼图中的"开发者入口"补齐。

### 4. AI 版支付宝"阿宝"正式开启邀请内测，超级 App 启动 Agent 重构
- **要点**：蚂蚁集团把 AI Agent 直接接入 10 亿月活的国民支付入口。
- **详情**：6 月 16 日，蚂蚁集团正式推出 AI 版支付宝，命名"阿宝"。用户在支付宝 App 搜索"AI 版支付宝"输入邀请码，即可在首页底栏左侧看到圆形 AI 图标，点击或右滑进入新界面。AI 版仅有两个功能页，用对话式交互替代原有的服务九宫格，目标是把支付、出行、生活缴费、理财等长尾服务统一收敛为 Agent 调用。
- **影响**：与微信"智能体计划"形成正面竞争，超级 App 范式从"流量分发"进入"任务执行"，对独立 Agent 创业公司形成入口级压力。
