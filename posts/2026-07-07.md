## 🎯 Agent 重点

### 1. 腾讯发布混元 Hy3 正式版，元宝上线免费 Agent 功能
- **要点**：基座模型迭代重点强化 Agent 能力，同步在元宝端向 C 端用户免费开放。
- **详情**：腾讯（Tencent）7 月 6 日正式发布基座模型 **混元 Hy3**，为 4 月 23 日开源预览版的迭代版本。团队进一步扩大了后训练算力规模，提升数据质量与多样性，重点强化了 Agent 能力与幻觉控制。混元 Hy3 同步接入 AI 助手元宝，首次向 C 端免费上线 Agent 功能。当日腾讯港股收盘报 452 港元。
- **影响**：这是腾讯自今年重组大模型团队后首次以正式版形态推向消费级市场，也标志着 BAT 三家在"通用助手 + 免费 Agent"路线上正式补齐最后一块拼图。
- 🔗 [财新：腾讯发布混元 3 大模型正式版](https://www.caixin.com/2026-07-06/102461425.html)

### 2. 阿里整合 QoderWork/悟空/MuleRun，推出企业级 Agent 桌面
- **要点**：阿里在 AI 办公首轮竞争失利后，用组织重构和产品整合迎战第二波。
- **详情**：财新 7 月 2 日获悉，阿里巴巴（Alibaba）将以可编程 Agent 桌面应用 **QoderWork** 为基础，整合钉钉孵化的企业级 AI 工作平台 **悟空**、以及"自进化"AI 工作空间 **MuleRun**，推出面向企业生产力场景的新 AI 产品。新产品由悟空事业部 CEO 陈宇森负责。此前阿里在办公场景第一波 Agent 应用竞争中未能占据优势。
- **影响**：反映"通用助手 + 企业级 Agent 桌面"正在成为国内大厂标准打法，与豆包、千问关停 UGC 智能体的合规策略形成对照——头部厂商正在把 Agent 战场从 C 端拟人聊天转向 B 端生产力工具。
- 🔗 [财新：阿里整合三款智能体产品](https://www.caixin.com/2026-07-03/102460271.html)

### 3. ICML 2026 首尔开幕，Agentic AI 成年度核心议题
- **要点**：AI 三大顶会之一 ICML 2026 于 7 月 6 日至 11 日在首尔举行，Agent 相关论文数量创历史新高。
- **详情**：本届 ICML 收到 **23,918 份**投稿，录用规模及 Agent 议程均创纪录，共评出 **10 篇获奖论文**（2 篇杰出论文奖、1 篇杰出立场论文奖、5 篇荣誉提名、1 篇杰出立场荣誉提名、1 篇时间检验奖）。杰出论文由清华黄高团队等华人研究者领衔，两篇均聚焦扩散模型；DeepMind 的 **A3C（异步优势演员-评论家）** 算法获得时间检验奖。会议还设有 Agent Memory、Computer-Use Agent、Tool Use 等多个专题。
- **影响**：学术界正把 Agent 从"应用工程"提升为一级研究方向，与工业界的 Agentic AI 产品化步调形成呼应。
- 🔗 [36氪：ICML 2026 获奖论文揭晓](https://www.36kr.com/p/3883880502046980)
- 🔗 [ICML 2026 官方论文列表](https://icml.cc/virtual/2026/papers.html)

### 4. Anthropic Claude Sonnet 5 发布一周，agentic 基准表现追平 Opus
- **要点**：定位"最强执行 Agent"的中端模型 Sonnet 5 已在 Claude Code 与 Platform 全量上线。
- **详情**：Anthropic 于 6 月 30 日深夜发布 **Claude Sonnet 5**，声称是 Sonnet 系列 agentic 能力最强的模型，具备制定多步计划、调用浏览器/终端并自主运行的能力。Sonnet 5 在多个 agentic 编码基准上贴近 Opus 4.8，但价格约为其 60%。Anthropic 同步推出面向科研人员的工作台 **Claude Science**。目前 Sonnet 5 已被 TechCrunch 描述为"运行 Agent 最便宜的旗舰替代方案"。
- **影响**：agentic 模型正进入"性能相近、比价格与延迟"的第二阶段，中端模型的经济性可能重塑企业 Agent 部署选型。
- 🔗 [Anthropic 官方：Introducing Claude Sonnet 5](https://www.anthropic.com/news/claude-sonnet-5)
- 🔗 [TechCrunch：Sonnet 5 定位分析](https://techcrunch.com/2026/06/30/anthropic-launches-claude-sonnet-5-as-a-cheaper-way-to-run-agents/)
