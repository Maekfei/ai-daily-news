## 🎯 Agent 重点

### 1. 月之暗面（Moonshot AI）发布全球最大开源模型 Kimi K3
- **要点**：2.8 万亿参数、原生多模态、100 万 Token 上下文，主打智能体编程。
- **详情**：7 月 16 日晚，Kimi K3 正式上线，总参数量 **2.8T** 稀疏 MoE 架构，基于自研 **KDA（Kimi Delta Attention）混合线性注意力机制**与注意力残差技术，原生支持视觉理解，上下文窗口拉到 100 万 Token。官方给出的定位是"Open Frontier Intelligence"，综合智能水平在开源阵营中位列全球第三，逼近 Claude Fable 5、GPT-5.6 Sol。**完整权重将于 7 月 27 日前开源**，API 与开发者文档已同步上线。
- **影响**：这是全球首个进入 3T 参数量级的开放权重模型，直接把中美开源模型能力差距压缩到一个季度以内，也把"智能体编程 + 长程知识工作"作为主赛道摆到台面——K3 官网首页新标语即"专为智能体编程与知识工作打造"。

### 2. WAIC 2026 上海开幕，"智能体手机 + 十万卡集群"入选十大镇馆之宝
- **要点**：为期 4 天、展览面积首破 10 万平米，1100+ 企业参展。
- **详情**：2026 世界人工智能大会暨人工智能全球治理高级别会议 7 月 17 日在上海世博、张江、西岸"三地四馆"同步启幕，主题"智能伙伴，共创未来"。国家主席习近平将出席开幕式并发表主旨讲话。大会展览总面积首次突破 **10 万平方米**，**1100 余家**企业集中展示 **3000 余项**前沿技术；十大"镇馆之宝"涵盖智能体手机、国产十万卡 AI 超集群、行业智能体、AI 眼镜、启元 T1 人形机器人等。

### 3. Anthropic 推 Claude Code Ultra：多智能体舰队并行验证每个 Bug
- **要点**：`/code-review ultra` 云端派出评审 Agent 舰队，先"找"后"验"。
- **详情**：Anthropic 在 Claude Code 中正式推出 `/code-review ultra` 命令——**云端托管**的评审智能体舰队，对 PR diff 进行深度扫描，并让每个发现被**独立 Agent 二次验证**后再上报，避免误报噪声。整体流水线遵循"Find → Verify"两段式：一批 Agent 扇出找 Bug，另一批独立 Agent 复核，只把真实问题推到开发者面前。官方文档明确定位为合并前的"deep, multi-agent code review"。
- **影响**：这是继 Claude Code、Codex CLI 之后，Coding Agent 从"辅助写"进入到"独立审"阶段的一次范式升级——评审阶段从单 Agent 判断升级为多 Agent 交叉验证，长期看会显著抬升企业代码合入门槛。
- 🔗 [Claude Code Docs: Find bugs with ultrareview](https://code.claude.com/docs/en/ultrareview)
