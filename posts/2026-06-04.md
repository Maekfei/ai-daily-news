## 🎯 Agent 重点

### 1. 微软 Build 2026：发布 MAI 全系自研模型，宣告"Agent 优先"时代
- **要点**：微软（Microsoft）一口气发布 7 款自研基础模型，并将 Agent 提升为 Windows 一等公民运行时。
- **详情**：在旧金山召开的 Build 2026 大会上，微软 AI 负责人 Mustafa Suleyman 推出 **MAI 原生模型家族**：旗舰推理模型 **MAI-Thinking-1**（350 亿活跃参数、MoE 架构、128K 上下文，未使用任何第三方模型蒸馏）、代码模型 **MAI-Code-1-Flash**（已集成至 GitHub Copilot 和 VSCode）、**MAI-Image-2.5** 等图像/语音模型。CEO 纳德拉开场即表示"Windows 不再只属于人类，Agent 现在是运行时、工具链和分发模型中的一等公民"，并发布 ACS（Agent Client Schema）开源规范。
- **影响**：标志着微软在底层模型层面与 OpenAI"切割"加速，并把 Windows 重新定义为面向 Agent 的开发平台，对 Copilot 生态和企业级 Agentic AI 落地形成关键推动。
- 🔗 [腾讯新闻 · 一文读懂微软 Build 2026：Agent 优先时代到来](https://news.qq.com/rain/a/20260603A01OUL00)
- 🔗 [IT之家 · 微软发布首款高级推理模型 MAI-Thinking-1](https://www.ithome.com/0/959/018.htm)

### 2. OpenAI"Intelligence at Work"发布会：Codex 全面并入 ChatGPT
- **要点**：OpenAI 把面向开发者的 Codex Agent 能力整合进 ChatGPT，覆盖十亿级用户的桌面/移动/浏览器/Office 场景。
- **详情**：6 月 2 日 OpenAI 举办线上发布会"Intelligence at Work"（AI 上岗），宣布未来几周内 Codex 将进入 ChatGPT，提供 Excel、Slack、PowerPoint 等企业常用工具的 Agent 操作能力，并面向"白领六大职业"推出新工具集。这是 Codex 自研究预览以来最大规模的一次产品形态升级，把开发者工具的 Agent 内核转化为通用知识工作者的桌面伙伴。
- **影响**：ChatGPT 与 Claude 在 Agent 能力上的产品形态进一步收敛，工作流自动化的入口之争从"专属 Agent App"转向"超级助手内置 Agent"。
- 🔗 [新浪科技 · 一夜之间，ChatGPT 变成了第二个 Claude](https://finance.sina.com.cn/tech/roll/2026-06-03/doc-iniachns4170690.shtml)

### 3. Anthropic 秘密递交 S-1，估值约 9650 亿美元
- **要点**：Claude 母公司 Anthropic 6 月 1 日向 SEC 保密提交 S-1 注册声明，启动 IPO 进程。
- **详情**：根据 cnBeta 援引外媒报道，Anthropic 估值约 **9650 亿美元**，预计最快今年秋季挂牌，正式招股书将在上市前 6–9 个月公开；公司预计 2026 Q2 首次实现盈利。结合此前 Crunchbase 报道的 H 轮 650 亿美元融资（估值接近 1 万亿美元），Anthropic 已在估值上反超 OpenAI（8520 亿美元），成为最值钱的 AI 创业公司。SpaceX、OpenAI、Anthropic 三家有望在 2026 年内集体登陆资本市场。
- **影响**：Agent 和企业级 AI 的代表玩家进入资本市场后，估值锚和披露要求都将被重新校准，对国内"AI 六小虎"上市路径具有强参考意义。

### 4. 腾讯微信 AI 智能体进入内测，开放超十亿用户入口
- **要点**：腾讯微信内嵌 AI Agent 原型进入测试阶段，最快本月启动合规审批。
- **详情**：6 月 2 日《金融时报》报道，腾讯正在测试一款嵌入微信主入口的 AI 智能体原型，可调用聊天、支付、小程序等微信生态能力；消息公布当日腾讯港股股价单日涨幅一度超过 9%。这是国内超级 App 首次把 Agent 作为系统级入口推向 10 亿+ 用户。
- **影响**：相较欧美的"操作系统级 Agent"路线，国内大概率走"超级 App 级 Agent"路线，腾讯此次内测将直接影响支付宝、抖音、淘宝等头部应用 Agent 化节奏。
