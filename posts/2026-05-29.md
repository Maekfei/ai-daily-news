# 🤖 今日 AI 要闻 · 2026-05-29

> Anthropic 旗舰模型 Claude Opus 4.8 正式上线，编程与 Agent 能力刷新榜单；DeepSeek 组建 Harness 团队对标 Claude Code，国产 Agent 产品化进入加速期。

## 🎯 Agent 重点

### 1. Anthropic 发布 Claude Opus 4.8，强化 Agent 与编程能力
- **要点**：旗舰模型小步快跑，距上代 Opus 4.7 仅 43 天，定价不变但新增 Fast Mode 与 Dynamic Workflows。
- **详情**：Claude Opus 4.8 在 SWE-bench Verified 拿到 **88.6 分**（Opus 4.7 为 87.6），SWE-bench Pro 为 **69.2 分**，在独立的 GDPval-AA 真实工作榜单上以 **1890 Elo** 登顶；定价维持 **$5/$25 per M tokens**，新增 **Fast Mode**（$10/$50 per M，速度约 2.5×），并随版本上线 **Dynamic Workflows**（研究预览），可让单个任务协调数百个并行子 Agent 跑通超大规模代码迁移。模型同步在 Pro/Max/Team/Enterprise 客户端、Claude API 以及 AWS / GCP / Azure 三大云平台开放。
- **影响**：在 GPT-5.6 据传月内发布、DeepSeek 组建 Harness 团队的背景下，Anthropic 用更密的迭代节奏稳住编程与 Agent 心智，"模型 + Harness + 工作流"的捆绑成为前沿厂商主流策略。
- 🔗 [Anthropic 官方公告：Introducing Claude Opus 4.8](https://www.anthropic.com/news/claude-opus-4-8)
- 🔗 [DataLearner 中文解读 · 编程与智能体能力详解](https://www.datalearner.com/blog/anthropic-released-opus-4_8)

### 2. 昆仑万维发布 SkyClaw-v1.0，国产 Agent 模型杀入第一梯队
- **要点**：天工 AI 推出 Agent 专用模型 SkyClaw-v1.0 与轻量版 SkyClaw-v1.0-lite，定价约为头部模型的一半。
- **详情**：SkyClaw-v1.0 支持 **百万 token 上下文**，重点优化复杂工具调用、多轮任务执行、代码生成与文件处理，主打"原生 Agent 训练范式"——不堆参数、围绕真实智能体工作流做强化学习。轻量版 SkyClaw-v1.0-lite 延续同一训练框架，但成本进一步压低，瞄准中小开发者与 Agent 平台集成方。
- **影响**：在 DeepSeek、阿里 Qwen、Kimi、智谱接连发力 Agent 后，昆仑万维以"差异化训练范式 + 半价策略"切入闭源高端 Agent 市场，国内 Agent 模型供给侧竞争进入新阶段。
- 🔗 [中国新闻网：昆仑万维 Agent 模型闯入全球第一梯队 定价低至行业一半](https://www.chinanews.com.cn/cj/2026/05-26/10628814.shtml)

### 3. DeepSeek 组建 Harness 团队对标 Claude Code
- **要点**：DeepSeek 资深研究员陈德里证实，公司将组建 Harness 团队，对标 Anthropic 的 Claude Code，自研代码 Agent 产品。
- **详情**：DeepSeek 在官网、小红书、X 同步放出 **Harness 产品经理 / Harness 研发工程师** 招聘，岗位定位为"模型之外的 Agent 工程"——上下文管理、工具调用、Sandbox、长任务编排等。背景上，Anthropic Claude Code 上线 6 个月年化收入已突破 **10 亿美元**，DeepSeek 同周完成 V4-Pro API **永久降价至原价 1/4**（输入 $0.435 / 百万 token），从纯模型供应商向"模型 + Agent 产品"全栈延展。
- **影响**：标志中国头部模型公司从"卖 token"转向"做 Agent 产品"，与 Claude Code、Cursor、SkyClaw 等形成正面竞争，推动国内 Coding Agent 生态进入工程化阶段。

- 🔗 [新浪财经周报：DeepSeek V4-Pro 永久降价 75%，组建代码智能体团队](https://finance.sina.com.cn/wm/2026-05-24/doc-inhyxwhw4711558.shtml)
