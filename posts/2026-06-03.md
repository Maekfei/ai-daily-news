# 🤖 今日 AI 要闻 · 2026-06-03

> 微软 Build 2026 全面押注「Windows 即 Agent 平台」并端出自研 MAI 模型家族；OpenAI 时隔六年重启机器人业务，AI Agent 战火从云端烧向操作系统与物理世界。

## 🎯 Agent 重点

### 1. 微软 Build 2026：Windows 升级为 Agent 平台，开源 Agent Framework
- **要点**：微软把 Windows、Azure、GitHub Copilot 全线改造成 Agent 原生栈，并 MIT 协议开源 Windows Agent Framework。
- **详情**：在 6 月 2 日开幕的 Build 2026 主题演讲中，CEO Satya Nadella 宣布 Windows Agent Framework (WAF) 1.0 正式开源、Windows Agent Runtime (WAR) 进入 Windows 11 内核侧；同步发布 Azure Agent Mesh（多 Agent 编排）、Copilot Workspace 结束 beta，以及面向 Agent 的全新概念硬件平台 Project Solara。Build 2026 由微软官方实时博客全程直播，3 天会期超 50 场 Agent 主题分论坛。
- **影响**：操作系统厂商首次把"Agent"内化为系统调用层级原语，将明显抬高 OS 级 Agent 的开发体验下限，并对所有"PC Agent"创业公司形成生态压力。
- 🔗 [Windows 官方博客：Windows 作为可信 Agent 开发平台](https://blogs.windows.com/windowsdeveloper/2026/06/02/build-2026-furthering-windows-as-the-trusted-platform-for-development/)
- 🔗 [Microsoft Build 2026 实时博客](https://news.microsoft.com/build-2026-live-blog)

### 2. 微软发布自研 MAI 模型家族，Project Polaris 取代 GPT-4 Turbo 驱动 Copilot
- **要点**：微软首次在 Build 推出完整自研 MAI 系列，含旗舰推理模型 MAI-Thinking-1（350B 中等规模）与 MAI-Code-1-Flash（SWE-bench 51%）。
- **详情**：MAI 家族一共 7 款模型，覆盖推理、编程、语音、转写与图像。其中 Project Polaris（编程模型）将于 2026 年 8 月成为 GitHub Copilot 默认引擎，取代此前的 OpenAI GPT-4 Turbo；MAI-Code-1-Flash 已上线 VS Code 与 GitHub Copilot CLI，在 SWE-bench 上拿到 51% 通过率。微软 AI CEO Mustafa Suleyman 在 keynote 上强调"未蒸馏任何第三方模型"。
- **影响**：标志微软在大模型层从"OpenAI 经销商"转向"自研 + 多供应商"，对 OpenAI 的最大客户身份产生明显裂痕。
- 🔗 [Microsoft AI 官方：Build 2026 MAI Keynote 全文](https://microsoft.ai/news/microsoft-build-2026-mai-keynote-transcript/)
- 🔗 [IT之家：MAI-Thinking-1 中文解读](https://www.ithome.com/0/959/018.htm)

### 3. OpenAI 时隔六年重启机器人业务，组建 OpenAI Robotics 团队
- **要点**：6 月 1 日 Sam Altman 在 X 发布 OpenAI Robotics 招聘启事，正式重返实体机器人战场。
- **详情**：OpenAI 公开招募全栈硬件、运营、系统及机器学习工程师；联创 Greg Brockman 同步发文称"团队正在快速推进，目标是打造能在物理世界帮助人类的 AI"。短期方向为协助技术工人建设基础设施的协助型机器人，长期愿景是"人人拥有可完成实体任务的家用机器人"，技术底座将复用 Sora 系列世界模型与多模态推理能力。OpenAI 上一次涉足实体机器人是 2020 年解散原机器人团队。
- **影响**：与 Figure、1X、Physical Intelligence 等 humanoid 创业公司形成正面竞争，并把"具身智能 + 大模型"路线推上新一轮资本与人才争夺。
- 🔗 [新浪财经：OpenAI 时隔六年重启机器人业务](https://finance.sina.com.cn/tech/roll/2026-06-01/doc-inhzwyqq3913115.shtml)
- 🔗 [腾讯新闻：Sora 大脑加持，OpenAI 自己造机器人本体](https://news.qq.com/rain/a/20260601A08Q3P00)

### 4. OpenAI 把 Codex Computer Use 上线 Windows，加入移动端控制
- **要点**：OpenAI Codex 的 computer-use 能力首次正式登陆 Windows，并支持手机端发起远程控制。
- **详情**：6 月 2 日 OpenAI 在 X 公告，Codex CLI / Codex Cloud 的"计算机使用"功能在 Windows 平台 GA，新增手机移动控件，可在手机上调度桌面 Agent 完成多步骤任务。这是 Codex 自 4 月"Codex for almost everything"扩展后又一次重要平台拓展，与微软同日发布的 Windows Agent Framework 形成微妙的"合作 + 竞争"关系。
- **影响**：桌面 Agent 实操从 macOS 一家独大走向 Windows 同等支持，叠加移动端遥控，让 Codex 成为可遥控的"远程开发员"。
- 🔗 [OpenAI 官方推文](https://x.com/OpenAI/status/2060428604727771421)
