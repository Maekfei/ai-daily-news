# 🤖 今日 AI 要闻 · 2026-06-14

> 智源大会发布"悟界"系列世界基座模型与 FlagOS 2.1，月之暗面同日开源 Kimi K2.7 Code 并推出全球首张 AI 原生信用卡，国产 Agent 生态密集落子。

## 🎯 Agent 重点

### 1. 月之暗面发布并开源 Kimi K2.7 Code，瞄准长程 Coding Agent
- **要点**：1.1 万亿参数代码专用模型同步开源，强化长程任务与 Agent 工作流。
- **详情**：月之暗面（Moonshot AI）于 6 月 13 日发布 Kimi K2.7 Code，**参数量 1.1T**、上下文窗口 **256K**，1M token 输入/输出价格维持 6.5 元 / 27 元，缓存命中输入降至 1.3 元；据官方内外部基准评测，模型在长上下文编程指令遵循、长程任务连贯性以及"过度思考"控制上均显著优于 K2.6。下周一还将上线 **6× 高速版（6x High-Speed Mode）**。
- **影响**：在 Anthropic Claude Code、OpenAI Codex 持续吸量的背景下，国产开源 Coding Agent 模型重新逼近第一梯队，给 Cursor / Cline / Roo Code 等开源 Agent 框架提供了新底座。
- 🔗 [OSCHINA：Kimi K2.7 Code 发布并开源](https://www.oschina.net/news/457565)

### 2. 智源研究院发布"悟界"系列：含全球首个通用世界基座模型 Physis-v0.1
- **要点**：智源一次性甩出 **3 个大模型 + 4 个智能体**，并升级开源 AI 系统软件栈 FlagOS 2.1。
- **详情**：在 6 月 12 日开幕的第八届北京智源大会上，智源研究院院长王仲远发布原生多模态大模型 **悟界·Emu3.5**、AI 药物发现模型 **悟界·OpenComplex**，以及全球首个通用世界基座模型 **悟界·Physis-v0.1** —— 后者从"预测下一个 token"升级到"预测下一个物理状态"，被视为具身智能的关键基础设施。同时发布配套的 4 个智能体与 FlagOS 2.1 软硬件生态，覆盖具身、科学、安全等场景。大会汇聚 14 位图灵奖得主历届出席名单中的 Whitfield Diffie 与 Andrew Barto 等 200 余位学者。
- **影响**：国内首次系统化对标 Google DeepMind Genie / NVIDIA Cosmos 路线，给国产具身智能与机器人产业链提供统一基座模型。
- 🔗 [中新网：北京智源大会开幕 发布通用世界基座模型等系列创新成果](https://www.chinanews.com.cn/cj/2026/06-13/10639624.shtml)
- 🔗 [搜狐：从"预测下一个词"到"预测下一个物理状态"](https://www.sohu.com/a/1035800066_115433)

### 3. MiniMax 开源 M3 旗舰多模态模型，华为云 / 摩尔线程 Day-0 适配
- **要点**：428B 总参 / 23B 激活的 MoE 多模态模型权重正式开放，国产算力同步打通。
- **详情**：MiniMax（稀宇科技）于 6 月 12-13 日正式开放 **M3 模型权重**（总参 ~428B、激活 ~23B），主打 **1M 上下文 + 原生多模态 + Coding Frontier** 三件套，SWE-Bench Pro 编程评测得分 **59.0%**，超越 GPT-5.5 与 Gemini 3.1 Pro。华为云 CloudMatrix（昇腾算力底座）完成开源首发适配并提供 Tokens 算力支持，**摩尔线程 MTT S5000** 同步完成 Day-0 适配。
- **影响**：国产 GPU + 国产开源大模型协同链路全面跑通，给企业私有化部署 Agent 工作流提供了新的"非英伟达"选项。
- 🔗 [新浪财经：MiniMax M3 发布并开源 摩尔线程 MTT S5000 完成 Day-0 极速适配](https://finance.sina.com.cn/tech/roll/2026-06-13/doc-inicfeqy9859708.shtml)
- 🔗 [IT之家：华为云与 MiniMax M3 实现开源首发适配](https://www.ithome.com/0/963/764.htm)

### 4. Kimi 推出全球首张 AI 原生信用卡，消费即转化为 Agent 算力额度
- **要点**：月之暗面把 Agent 调用额度做成可流通的"消费权益"。
- **详情**：6 月 12 日，Kimi 宣布全球首张 **AI 原生信用卡**开启预约，每一笔消费都会按比例转化为 **AI 算力额度**，可直接兑换 Agent 使用次数、高级功能权限等；目前已与一家国有大行及国际卡组织合作，正在探索支付与 AI 服务结合的新场景。
- **影响**：标志着 AI Agent 调用量首次进入主流支付权益体系，对 OpenAI ChatGPT、Anthropic Claude 等订阅制模式形成差异化挑战，也为后续 "Agent 即服务"按算力计费铺路。
- 🔗 [腾讯新闻：每一笔消费均可转化为 AI 算力额度 Kimi 全球首张 AI 原生信用卡开放预约](https://news.qq.com/rain/a/20260612A06G5G00)
