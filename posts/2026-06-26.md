### 1. 中科闻歌登陆港交所，"通用决策大模型第一股"开盘大涨
- **要点**：港股公开发售获 5966 倍超额认购，AI 国家队首次叩开二级市场。
- **详情**：6 月 26 日，北京中科闻歌科技股份有限公司（股票代码 **01956.HK**）正式在港交所主板挂牌上市。发售价每股 **60.70 港元**，全球发售 1483 万股，募资净额约 **8.27 亿港元**，IPO 估值约 **105.1 亿港元**。公司有中科院背景加持，长期聚焦复杂数据分析与人工智能辅助决策，自定位"通用决策大模型"赛道。中科闻歌香港公开发售部分录得 **5966 倍** 超购，超过此前 MiniMax（1209 倍）、智谱（1164 倍）的纪录；暗盘交易涨幅近 95%。基石投资者包括中国东方、嘉实国际、前海国际等机构，合计认购 3100 万美元。
- **影响**：在 DeepSeek 完成 500 亿元融资、Anthropic 估值飙至 9650 亿美元的产业背景下，国内"决策类 Agent"概念股获得首次完整估值锚定，为后续智谱、MiniMax 等独角兽的资本化路径提供新参照。

### 2. Patronus AI 完成 5000 万美元 B 轮，发布首个"数字世界模型"训练 Agent
- **要点**：累计融资达 7000 万美元，过去一年营收增长 **15 倍**，押注大规模智能体仿真。
- **详情**：6 月 25 日，专注 AI 评估与仿真的初创公司 **Patronus AI** 宣布完成 **5000 万美元 B 轮融资**，由 Greenfield Partners 领投，Notable Capital、光速创投、Datadog、三星等老股东及多位 AI 行业高管跟投。公司同步推出业内首款"**Digital World Model**"——一个能预测和模拟智能体在数字工作流中行为的大规模仿真环境，让 AI Agent 在部署到真实业务系统前进行压力测试。Patronus AI 由前 Meta AI 研究员 Anand Kannappan 与 Rebecca Qian 于 2023 年联合创立，已服务多家头部 AI 实验室与金融、医疗客户。
- **影响**：Agent 落地从"能不能跑"进入"敢不敢上线"阶段，Patronus 把测试范式从静态 benchmark 推进到沙盒化的"数字孪生世界"，预示 Agent 安全/可靠性赛道将形成独立基础设施层。

### 3. Anthropic 发布 Claude Tag：让 Claude 以"虚拟员工"身份常驻 Slack
- **要点**：Claude Tag 引入 Agent 独立身份与权限模型，企业内部 65% 代码已由其生成。
- **详情**：北京时间 6 月 24 日，**Anthropic** 正式发布 **Claude Tag**，一款常驻 Slack 频道的智能 AI 协作工具，目前以 Beta 形式面向 Claude Team / Enterprise 用户开放。Claude Tag 基于现有 Claude Code 与 Claude Cowork 构建，支持"多用户协作"与"智能代理"两种模式：在环境行为模式下能主动观察频道、跟进未完成任务并异步交付。最大创新是为每个 Tag 实例配置 **独立身份、工具权限与 Token 预算**，与传统"用户授权 AI"的模型彻底分开；Anthropic 自家产品团队披露 **65% 的代码** 由内部 Claude Tag 生成。
- **影响**：Anthropic 给"Human-Agent Teaming"提供了首个产品化样板——AI 不再寄生在某个员工账号下，而是以可审计的数字同事身份进入企业组织架构，推动智能体安全治理范式落地。
- 🔗 [搜狐：Anthropic 推 Slack 版 Claude Tag 企业用率首超 OpenAI](https://www.sohu.com/a/1040971008_120333371)
- 🔗 [aihub.cn：Claude Tag 产品页](https://www.aihub.cn/agents/claude-tag/)

### 4. Google DeepMind 把 Native Computer Use 集成进 Gemini 3.5 Flash
- **要点**：Gemini 3.5 Flash 原生支持桌面与移动设备操作，企业级安全沙盒同步发布。
- **详情**：6 月 25 日，Google DeepMind 宣布在 **Gemini 3.5 Flash** 模型中原生集成 **Computer Use** 能力，AI Agent 可直接驱动桌面浏览器、移动 App 与 SaaS 工具完成端到端任务，无需额外的工具调用层。本次更新还包括企业级安全增强（细粒度权限审计、流量隔离），针对客服、数据录入、跨系统流程自动化等高频场景。
- **影响**：在 Anthropic Claude Computer Use、OpenAI Operator 之后，Google 把 computer use 能力下沉到中等性价比的 Flash 档位，企业大规模部署成本进一步下降。

### 5. 新华社"新华语典"权威 AI 时政智能体正式上线
- **要点**：主流媒体首个"主流价值观驾驭"的时政 Agent 上线，背后投资 11.2 亿元、已通过中信通院验收。
- **详情**：6 月 26 日，新华社打造的权威 AI 时政资讯智能体"**新华语典**"正式上线。项目此前已通过中国信通院组织的验收评审，专家来自中国工程院、中央党校、中宣部、清华大学等。新华网今年早些时候将原计划用于全媒体云平台、移动内容分发等四个传统信息化项目的 **7.4 亿元募资** 调整投向"新华语典"，叠加既有预算总投入达 **11.2 亿元**。智能体定位为"学习研究传播党的创新理论的智能体""传播权威时政新闻的智能体"，已在重庆、内蒙古兴安盟等地开展政务场景应用推广。
- **影响**：国资主流媒体首次大规模落地 Agent 级产品，意味着官方信源对接 AI 检索/生成的合规路径开始打通，对国内政务、舆情、合规类 Agent 应用有直接示范意义。
