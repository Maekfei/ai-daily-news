# 🤖 今日 AI 要闻 · 2026-07-21

> WAIC 2026 收官次日，AI Agent 生态进入「基础设施 + 出海」双主线：英伟达把 Agent 塞进 DGX Station 本地工作站，阿里云 Agent Native Cloud 与腾讯 ADP 4.0 海外版正面开火。

## 🎯 Agent 重点

### 1. NVIDIA（英伟达）推出 Agent Toolkit：DGX Station 本地 30 分钟部署智能体
- **要点**：英伟达把「个人 AI 智能体」下沉到桌面级超算，主打隐私可控与本地推理。
- **详情**：7 月 20 日，NVIDIA 面向搭载 **GB300 Grace Blackwell Ultra** 桌面超级芯片的 **DGX Station** 工作站推出 **Agent Toolkit** 软件工具包。该套件基于官方 build.nvidia.com/station 页面提供 NemoClaw 沙箱、可选 Telegram 交互接口，用户可在约 **30 分钟** 内本地部署个人编程/助理 Agent，DGX Station 单机可承载 **万亿参数** 级模型、FP4 精度下达 **1 PFLOPS** AI 性能。该动作与英伟达在 GTC 2026 上宣告的从「Scaling Law」到「Agentic Scaling」范式转型一脉相承。
- **影响**：这是首个把 Agent 部署门槛压到「一台桌面工作站 + 半小时」的官方方案，为受限于数据出域的金融、医疗、政务客户提供了 SaaS Agent 之外的第二条路径，也把 Anthropic、OpenAI 依赖云端订阅的商业模式推向短兵相接。
- 🔗 [Wccftech：NVIDIA Brings Local AI Agents To DGX Station With Agent Toolkit](https://wccftech.com/nvidia-brings-local-ai-agents-to-dgx-station-pc-with-agent-toolkit/)
- 🔗 [NVIDIA 官方：build.nvidia.com/station](https://build.nvidia.com/station)

### 2. 阿里云在 WAIC 发布 Agent Native Cloud：智能体成为企业「原生」能力
- **要点**：阿里云正式提出「智能体原生云」概念，把 Agent 从工具升级为云基础设施的一等公民。
- **详情**：7 月 18 日 WAIC 2026 期间，阿里云发布 **Agent Native Cloud**，同步推出 **AgentTeams**（多智能体协同）、**Agentic Computer**（云桌面级 Agent 载体）等企业级产品，覆盖基础设施、开发平台、云桌面三层。阿里云云原生应用平台负责人周琦提出核心判断：「下一轮竞争，比的不是谁拥有更多 Agent，而是谁能把 Agents 变成可控、可复用、可协作、会进化的组织资产。」产品体系围绕 Agent 生命周期打造，与阿里云在 5 月峰会已完成的「芯片-云-模型-推理」全栈 Agent 化升级形成闭环。
- **影响**：继微软 Build 2026 Agent Platform、AWS Bedrock AgentCore 后，中国云厂商正式给出 Agent 原生的完整技术栈定义。国内 Agent 基础设施赛道进入「三巨头 + 字节火山方舟」四强格局。
- 🔗 [腾讯网：阿里云发布 Agent Native Cloud 全栈产品体系](https://news.qq.com/rain/a/20260719A042XY00)
- 🔗 [环球网科技：2026WAIC 阿里云发布 Agent Native Cloud](https://tech.huanqiu.com/article/4SQj8HxdmyN)

### 3. WAIC 2026 闭幕次日盘点：AI 从「能对话」全面走向「会干活」
- **要点**：官方媒体总结本届 WAIC 三大主线，「智能伙伴」取代「大模型军备」成为行业共识。
- **详情**：7 月 17 日至 20 日，2026 世界人工智能大会（WAIC）在上海举行，总展览面积超 **10 万平方米**，**1100+** 家企业携 **3000+** 项展品参展，**300+** 款产品全球首发。新华社在 7 月 20 日盘点稿中勾勒三大发展主线：一是 AI 从「工具」向「智能伙伴」跃迁；二是具身智能规模化商用（60 台人形机器人上岗做导览）；三是国产算力集群与端侧模型合规落地形成产业闭环。**蚂蚁灵波「一脑多机」智慧药房**入选十大镇馆之宝，机器人在国大药房实现「问诊-购药-处方流转-取药-打包」**90 秒** 全流程；启元 T1 变形机器人首次公开亮相。
- **影响**：这是 WAIC 九届以来规模最大的一届，参展企业数与首发产品数均创新高。中国 Agent 与具身智能赛道从「概念秀场」进入「千行百业落地兑现期」，为下半年一级市场估值锚点提供了明确信号。

### 4. 腾讯云 ADP 4.0 海外版正式发布：中国 Agent 平台联手出海东南亚
- **要点**：腾讯云 Agent Development Platform 4.0 海外版在 WAIC 期间正式对外，主攻中企出海东南亚场景。
- **详情**：7 月 18 日，腾讯云副总裁吴运声在 WAIC 主论坛发布 **ADP 4.0 海外版**，通用桌面智能体 **WorkBuddy App 版本** 同步上线。ADP 4.0 提供从智能体构建、编排、评测到运维的全链路能力，原生兼容 **MCP** 与 **A2A** 协议，可对接外部工具与第三方 Agent。WorkBuddy 则把桌面级 Agent 打包为开箱即用的生产力工具，主打 AI Coding、文档处理、客户服务。腾讯同期展台以「Hey，我的 AI Buddy」为主题，集中展示 AI 从「能说」走向「会干」的落地案例。
- **影响**：继阿里云 Agent Native Cloud 之后，腾讯给出中国云厂商「Agent 出海」的第二个完整答卷，直指印尼、泰国等数字金融爆发市场，与 AWS Bedrock AgentCore、Google Cloud Vertex Agents 形成正面竞争。
