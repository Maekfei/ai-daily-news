> SpaceX 以 600 亿美元换股收购 AI 编程独角兽 Cursor，G7 峰会三大 AI 巨头掌门人首次同台讨论 AI 治理。

## 🎯 Agent 重点

### 1. SpaceX 以 600 亿美元换股收购 AI 编程公司 Cursor
- **要点**：SpaceX 上市后首单并购，将 Anysphere 整建制纳入 xAI 生态，剑指企业级 AI 编程市场。
- **详情**：SpaceX 6 月 16 日向 SEC 提交合并协议，全资子公司 X67 将与 Anysphere（Cursor 母公司）合并，Cursor 整体估值约 600 亿美元，全部以 SpaceX A 类普通股置换、不涉及现金支付。Cursor 股东将按 SpaceX 前 7 个交易日均价获得对应股份，交易预计 2026 年三季度完成，仍待监管审批。这是 SpaceX 上市后首单并购，市场普遍解读为马斯克借此补足 xAI 在 AI Coding 赛道的短板，直接对标 GitHub Copilot 与 Claude Code。
- **影响**：标志着 AI 编程 Agent 赛道从"创业公司互卷"进入"巨头整合期"，Cursor 与 Grok / xAI 模型栈的协同将重新洗牌开发者工具市场格局。
- 🔗 [财新 · SpaceX 600 亿美元换股收购 Cursor](https://www.caixin.com/2026-06-17/102454873.html)

### 2. 阿里发布千问首个具身智能大模型系列 Qwen-Robot
- **要点**：一口气推出操作、导航、世界三大模型，覆盖机器人"手、脚、脑"全栈。
- **详情**：6 月 16 日，阿里巴巴正式发布 Qwen-Robot 系列具身智能大模型，包含 VLA 操作模型 Qwen-RobotManip、VLN 移动模型 Qwen-RobotNav、世界模型 Qwen-RobotWorld 三款。三个模型既可独立部署也能协同运转，分别对应机器人的精细操作、空间导航与认知规划能力，意在解决异构机器人本体的适配难题。这是千问大模型家族首个完整的具身智能模型矩阵。
- **影响**：与近期千寻智能、星动纪元等创业公司的密集融资形成共振，国内具身智能基础模型层进入"大厂下场"阶段。
- 🔗 [第一财经 · 阿里发布首个具身大模型 Qwen-Robot 系列](https://www.yicai.com/news/103232043.html)

### 3. 智谱开源旗舰模型 GLM-5.2，主打 Coding 与长程 Agent
- **要点**：744B 总参 / 40B 激活 MoE 架构，1M 上下文，MIT 协议开源，直接对标 Claude Opus 编码能力。
- **详情**：6 月 17 日智谱 AI 正式上线并开源 GLM-5.2，重点优化长上下文 Coding、长程 Agent 任务执行与复杂工程项目能力。模型采用 MIT 开源协议，支持 1M token 上下文窗口，已面向 GLM Coding Plan 全量用户开放。背景是 Anthropic Claude 因美国出口管制在中国全面下线后，智谱抓住窗口期承接企业级 Coding Agent 需求。
- **影响**：国产开源 LLM 进一步逼近闭源前沿模型，企业开发者在 Cursor / Claude Code 之外多了一条可控的开源 Agent 基座选项。
- 🔗 [AIHub · 智谱 GLM-5.2 上线并开源](https://www.aihub.cn/news/glm-5-2-open-source/)

### 4. 月之暗面 Kimi K2.7 Code 高速版上线，最高 6 倍输出速度
- **要点**：1.1 万亿参数编程模型推出加速档，2 倍价格换取 5-6 倍速度。
- **详情**：6 月 15 日月之暗面宣布 Kimi K2.7 Code 高速版正式上线 API，与原版同模型但输出速度提升 5-6 倍，常规编程场景输出速度约 180 Tokens/s，短上下文最高可达 260 Tokens/s，价格为普通版的 2 倍。该模型已于 6 月 12 日开源，相比 K2.6 在长上下文指令遵循、长程编程任务表现上明显提升，token 平均消耗降低 30%。已向 Kimi Code Beta、API 开发者及 Kimi Business 用户开放。
- **影响**：在 Claude Code 与 Codex 同质化竞争白热化的背景下，国内编程 Agent 模型以"速度+性价比"切入开发者工具市场，加大了 Cursor 等套壳类产品的基座切换灵活度。
- 🔗 [新浪科技 · Kimi K2.7 Code 高速版上线](https://finance.sina.com.cn/tech/digi/2026-06-15/doc-inicpewk6604233.shtml)
