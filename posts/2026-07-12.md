## 🎯 Agent 重点

### 1. Anthropic 为桌面版 Claude Code 内置沙箱浏览器，Agent 编程闭环再进一步
- **要点**：Claude Code Desktop v1.2581.0 新增内置浏览器（Cmd+Shift+B），Agent 可自主读页、点击、填表、截图，编程闭环从「写代码」延伸到「验证代码」。
- **详情**：**7 月 10 日**（美国当地时间），**Anthropic**（Claude 母公司）官方账号 **@ClaudeDevs** 在 X 平台宣布：桌面版 **Claude Code** 新增**沙箱化应用内浏览器**，开发者按 **Cmd+Shift+B** 即可在 Agent 会话中打开网页——**读文档、点按钮、填表单、抓取截图**全部在一个进程内完成，且无需切换到 Chrome。功能定位直接对标 **OpenAI Codex 内置浏览器**与 **Browserbase** 云 Agent 浏览器；结合 **7 月 3 日 GitHub Copilot 首次接入开源模型 Kimi K2.7 Code**（1 万亿参数、价格为 Claude Opus 4.8 的 1/5）的行业背景，Agent 编程赛道正在同时被闭源龙头「加护城河」与开源新贵「拆护城河」两端拉扯。
- **影响**：把 Web 交互沉到 Agent runtime 内是 Anthropic 押注的关键路径——未来 Agent 不再需要外挂 Playwright/Puppeteer，Session/Sandbox/Browser 三合一将成为下一代 Agent OS 的标配；对开发者来说，QA/E2E 测试、爬虫脚本、可视化调试的工作流将被显著压缩。

### 2. OpenAI 发布 ChatGPT Work：GPT-5.6 驱动的企业 Agent 平台上线
- **要点**：OpenAI 在 GPT-5.6 全量开放的同一时点推出 **ChatGPT Work**，用自然语言指令自动操作邮件、Slack、日历、Google Docs、Sheets、Sites 等第三方业务应用。
- **详情**：**7 月 9-10 日**，**OpenAI** 官宣 **ChatGPT Work**——一款云端 **Agentic 平台**，可跨邮件、Slack、日历、Google Workspace 等应用执行多步任务，首发限定 **Pro / Enterprise / Edu** 用户，Plus 和 Business 「未来几天」跟进；底层模型为同期 GA 的 **GPT-5.6** 系列（**Sol / Terra / Luna** 三档），最高支持 **1.5M token 长上下文**与 **Realtime Voice 2.0**。这是 OpenAI 首次把「Agent 模式」下沉到订阅账户的默认能力集，被外界解读为其冲击 IPO 前的关键「企业变现」证据链——需要证明其消费级用户能被转化为可持续企业 ARR。
- **影响**：叠加同期 **Anthropic ChatGPT Work / Claude for Enterprise** 与 **Google Gemini Managed Agents** 的密集升级，企业 Agent 平台竞争在一周内从「炫技」转向「谁能接管更多 SaaS 工作流」；Salesforce、Workday、ServiceNow 等 SaaS 厂商的 Agent 中间层地位承压。

### 3. Google 扩展 Gemini API Managed Agents：新增后台执行 + 远程 MCP Server
- **要点**：Gemini API Managed Agents 一次性引入**后台任务、远程 MCP Server 集成、自定义 function calling、上下文刷新**四项能力，直接把 MCP 生态并入 Gemini 官方 Agent 运行时。
- **详情**：**7 月 7 日**，Google 在开发者博客宣布 **Gemini API Managed Agents** 大版本升级，核心变化：（1）**后台执行**：长任务不再需要客户端保持连接；（2）**远程 MCP Server 集成**：直接把用户已部署的 MCP Server 注册进 Managed Agent，无需搭桥；（3）**自定义 function calling** 与 refreshing 支持；（4）通用型 **Antigravity Agent**（`antigravity-preview-05-2026`）进入 **public preview**，可自主规划、编写并执行代码。此次更新距 **MCP 2026-07-28 规范正式发布**（无状态化、OAuth 2.1、Extensions 框架）不到 3 周，属于 Google 在 MCP 新规上的抢跑站队。
- **影响**：Google 是继 Anthropic 之后第二家把 MCP 深度接入自家 Managed Agent 平台的一线大厂；「MCP as the Agent USB-C」的行业共识虽在 Perplexity 于 3 月「弃 MCP 转 API+CLI」时出现分裂，但 Google 的入场再度巩固了 MCP 作为跨平台 Agent 工具协议的地位。
