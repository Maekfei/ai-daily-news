# 🤖 今日 AI 要闻 · 2026-05-28

> Anthropic 即将以 **9000 亿美元**估值收尾新一轮 300 亿美元融资，超越 OpenAI 登顶全球 AI 初创估值榜首；阿里云 Qwen Cloud 正式出海，把"Agent 时代的云入口"摆到全球开发者面前。

## 🎯 Agent 重点

### 1. Anthropic 即将以 9000 亿美元估值收尾 300 亿美元巨额融资
- **要点**：估值有望反超 OpenAI 8520 亿美元，Anthropic 借 Agent + 编码场景登顶全球 AI 初创榜首。
- **详情**：据彭博与多家媒体 5 月 26–27 日报道，Anthropic（Claude 母公司）即将完成新一轮规模超过 **300 亿美元**的融资，**估值突破 9000 亿美元**，超越 OpenAI 当前 8520 亿美元的私募估值。本轮由红杉资本、Dragoneer、Altimeter、Greenoaks 联合领投，Peter Thiel 旗下 Founders Fund 也参与跟投。新资金将主要投入算力采购（与 AWS Trainium、Google TPU 形成多源供给）以及 Claude Code、Mythos 系列前沿模型的训练。同期消息称，Anthropic 也在评估最快 10 月启动 IPO 的选项。
- **影响**：私募市场用真金白银把"企业 Agent + 编码"赛道定价为首位，Cursor、Cognition、Replit 等被视为 Anthropic 体系内最受益的中间层；对 OpenAI 而言，紧迫的 IPO 文件提交（据传未来数周）压力进一步放大。
- 🔗 [每日经济新闻：Anthropic 最快本周完成超 300 亿美元融资，估值有望破 9000 亿美元](https://www.nbd.com.cn/articles/2026-05-25/4406727.html)

### 2. 阿里云 Qwen Cloud 出海，全栈 Agent 云入口同步登场
- **要点**：阿里云面向海外开发者推出全新 AI 产品官网，定位为"为 Agent 时代设计的云入口"而非传统控制台子模块。
- **详情**：5 月 26 日阿里云宣布上线 **qwencloud.com**——继国内"千问云"之后面向海外的全新 AI 产品门户。CTO 周靖人表示，海外市场对 Agent 的需求让模型调用量和云资源消耗呈**指数级增长**，Qwen Cloud 把模型（Qwen3.7-Max / Qwen3-VL / Qwen-Coder）、Agent 产品（QoderWake、Qwen Agent OS）、云基础设施（GPU + 真武 M890 训推一体芯片）做统一打包。开发者可一键拉起多 Agent 编排任务，且按 token 与 Agent-step 双维度计费。
- **影响**：阿里云延续"全栈出海"策略，与 AWS Bedrock、Azure AI Foundry、Google Vertex 直接竞争开发者心智；与 Qwen3.7-Max 推理性能跻身全球第一梯队叠加，国产模型的海外分发渠道被显著缩短。
- 🔗 [阿里云官方：Qwen Cloud 上线，加速 AI 出海](https://www.alibabacloud.com/blog/qwen-cloud-launch)

### 3. Fireworks AI 估值四倍跳涨至 150 亿美元，推理基础设施再受热捧
- **要点**：彭博披露 Fireworks AI 正洽谈以 **150 亿美元**估值融资，较 2025 年 10 月的 40 亿美元上涨近 4 倍。
- **详情**：Fireworks AI 是企业级 AI 推理加速平台，主力产品 FireOptimizer / FireFunction 帮助客户在自有云或多云环境跑 Llama / Qwen / DeepSeek 等开源模型，主打 **TTFT < 100ms、吞吐高 3–5 倍**的高速推理。彭博 5 月 27 日报道称，本轮融资由顶级成长基金参与，公司年化收入据称已超过 1.5 亿美元，企业客户包括 Cursor、Notion、DoorDash 等。该轮估值跳涨主因是 Agent 工作负载推动**托管推理需求井喷**，同位竞争者还包括 Together AI、Anyscale、Replicate。
- **影响**：在 OpenRouter 1.13 亿美元 B 轮（昨日）之后，"模型路由层 + 推理执行层"两条赛道同时被资本明确定价；闭源大模型商对 token 单价的话语权进一步松动。
- 🔗 [Bloomberg Law: Fireworks AI In Talks for Funding at $15 Billion Valuation](https://news.bloomberglaw.com/private-equity/fireworks-ai-in-talks-for-funding-at-15-billion-valuation)

### 4. xAI 推出 Grok Skills，把"持久记忆 + 可编程工作空间"塞进 Agent
- **要点**：xAI 在网页、iOS、Android 三端同步上线 **Skills**，让 Grok 从单次问答聊天机器人升级为"可配置自动化工作空间"。
- **详情**：5 月 19 日—27 日间逐步全量推送的 **Grok Skills** 给 AI 助手 Grok 注入了跨对话的**持久记忆**与**可复用技能模板**：用户可像 ChatGPT GPTs 一样把"研究助手 / 邮件代写 / 投研抓取"等工作流封装成 Skill，并附带自定义工具调用、知识库与触发条件。Skills 与 Grok 的实时 X（Twitter）数据流深度耦合，这是 OpenAI / Anthropic 难以复刻的差异化优势。
- **影响**：xAI 对标的不是 ChatGPT 而是 ChatGPT + Operator + Memory 的合集；在 Grok V9 模型曲线趋于持平的背景下，xAI 选择从 Agent 框架与差异化数据切入战场。
- 🔗 [界面新闻：xAI 推出 Skills，Grok 走向可编程工作空间](https://www.jiemian.com/article/12195133.html)
