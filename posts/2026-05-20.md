## 🎯 Agent 重点

### 1. Google I/O 2026：发布 Gemini Spark 全天候个人 AI 智能体
- **要点**：谷歌正式把"对话式 AI"推到"长时运行的代理"时代，Gemini Spark 在 Google Cloud 虚拟机里 24/7 替用户跑活。
- **详情**：北京时间 5 月 20 日凌晨，Sundar Pichai 在 Google I/O 2026 主题演讲中发布 Gemini Spark——基于 Gemini 3.5 基础模型与一套全新 "agentic harness"，能在云端虚拟机里持续运行，主动起草邮件、整理收件箱、跨应用下单购物，并与 Gmail、Calendar、Drive、Maps 深度集成。Spark 通过 goal persistence + task decomposition + tool orchestration 的机制保持长时记忆与状态恢复，开发者可通过 Vertex AI 接入同一 harness 构建自家代理。
- **影响**：这是谷歌首次把"代理"作为消费级产品形态而非实验室 demo 推向数十亿 Gmail 用户，直接对位 Anthropic 的 Computer Use 与 OpenAI 的 Operator/ChatGPT Agent，标志 Big Tech 集体进入"Daily Active Agent"竞争。
- 🔗 [TechCrunch：Google introduces Gemini Spark, a 24/7 agentic assistant with Gmail integration](https://techcrunch.com/2026/05/19/google-introduces-gemini-spark-a-24-7-agentic-assistant-with-gmail-integration/)
- 🔗 [Mashable：Google I/O 2026: Gemini Spark is a wildly ambitious AI agent](https://mashable.com/article/google-io-2026-gemini-spark-announced)

### 2. Anthropic 斥资逾 3 亿美元收购 Stainless，握紧 AI SDK 命门
- **要点**：Anthropic 把 OpenAI、Google、Cloudflare 都在用的 SDK 自动化工具公司收入囊中，配合自家发明的 MCP 协议，进一步把"代理连接性"垄断成自家护城河。
- **详情**：5 月 18 日，Anthropic 宣布收购总部位于纽约的开发者工具初创 Stainless，由前 Stripe 工程师 Alex Rattray 创办，其工具被 OpenAI、Google、Cloudflare 等竞争对手大规模用于自动生成 API SDK。Analytics Insight 与 IT 之家援引行业消息称交易额超 **3 亿美元**（IT 之家口径 2.8 亿欧元）。Stainless 团队整建制并入 Anthropic Developer Platform，托管类产品将逐步关闭，已签约客户的 SDK 生成服务保留至到期。
- **影响**：Anthropic 既是 MCP 协议的发起方，又收紧上游 SDK 工具链，相当于把"代理与外部世界握手"的两端都纳入自家版图——竞品要么自建、要么继续付费给 Anthropic 系生态，对手 Google/OpenAI 的工程效率短期会受影响。
- 🔗 [Anthropic 官方公告：Anthropic acquires Stainless](https://www.anthropic.com/news/anthropic-acquires-stainless)
- 🔗 [TechCrunch：Anthropic has acquired the dev tools startup used by OpenAI, Google, and Cloudflare](https://techcrunch.com/2026/05/18/anthropic-has-acquired-the-dev-tools-startup-used-by-openai-google-and-cloudflare/)
- 🔗 [Analytics Insight：Anthropic Acquires Stainless for Over $300M](https://www.analyticsinsight.net/news/anthropic-acquires-stainless-for-over-300m-to-strengthen-ai-sdk-and-tool-access)

### 3. SandboxAQ 把药物发现模型接入 Claude，"无需博士"做药
- **要点**：Anthropic 推动垂类科学代理落地，SandboxAQ 的量子化学模型直接挂到 Claude 上，自然语言即可驱动新药筛选。
- **详情**：5 月 18 日，Alphabet 系拆分公司 SandboxAQ 宣布把其用于药物发现的计算化学模型集成进 Claude，研究者可通过自然语言对话调用分子模拟、结合能预测、ADMET 评估等流程，原本需要计算化学博士才能跑的工作流被压缩成 prompt 级操作。两家公司称首批生物制药客户已开始内测。
- **影响**：这是"行业模型 + 通用代理"模式的代表样本——专业模型不再追求自建对话产品，而是作为工具被通用代理调用，加速 LLM 在生命科学、材料、半导体 EDA 等垂类的渗透。
- 🔗 [TechCrunch：SandboxAQ brings its drug discovery models to Claude — no PhD in computing required](https://techcrunch.com/2026/05/18/sandboxaq-brings-its-drug-discovery-models-to-claude-no-phd-in-computing-required/)
