## 🎯 Agent 重点

### 1. Anthropic Claude Fable 5 / Mythos 5 遭美商务部出口管制全球下架
- **要点**：上线仅 **72 小时**，美国出口管制首次叫停商业前沿大模型，外籍员工亦被切断访问。
- **详情**：当地时间 6 月 12 日下午 5:21，Anthropic（Claude 母公司）收到美国商务部长卢特尼克签发的出口管制指令，要求暂停所有 **外国主体**（含美国境外机构、个人，以及境内全部外籍员工）对 **Claude Fable 5 与 Mythos 5** 的访问；据《华尔街日报》报道，这一禁令的导火索是 **Amazon CEO Andy Jassy** 向美国财政部报告 —— 亚马逊研究人员在内测中发现 Fable 5 可被用于获取**网络攻击相关信息**。Anthropic 当晚发长文驳斥并强调已采取「纵深防御」策略，但模型仍被强制下架。Fable 5 是 Anthropic Mythos 系列首款大规模发布模型，号称在软件工程、知识工作、视觉任务上「随任务复杂度提升优势越显著」。
- **影响**：这是首次有商业 AI 大模型因美国出口管制被全球下架，给海外企业的 Agent / Coding 工作流（Cursor、Claude Code、Cline、Roo Code 等）造成切换风险，国产大模型与闭源 OpenAI/Google 模型迎来阶段性替代窗口。

### 4. UC Berkeley 等推出「Agents' Last Exam」基准，最强模型仅 8.6%
- **要点**：250+ 行业专家共建的"经济价值 Agent 终极考"出炉，前沿模型集体翻车。
- **详情**：6 月 10 日前后，**加州大学伯克利分校** 联合超过 **250 位行业专家** 发布 Agent 评测新基准 **Agents' Last Exam（ALE）**，论文于 6 月 3 日上线 arXiv（编号 2606.05405）。ALE 用真实、长期、有经济价值的专业工作流任务作为试题，并采用基于产物（artifact-based）的严格评分；据 36氪 / 51CTO 报道，最强参赛模型最终得分率仅 **8.6%**，远低于此前 SWE-bench / GAIA 等基准给人的"接近人类"的乐观印象。
- **影响**：在巨头炒「Agent 元年」的当口，ALE 像是给行业泼了一盆冷水 —— 通用 Agent 距离真正替代知识工作者的差距，比传统 benchmark 显示的要大得多；可能成为下半年模型公司争抢的下一座灯塔。
