# 🤖 今日 AI 要闻 · 2026-06-22

> 《自然》同期发表的 MIRA 与谷歌 AMIE 两套自主医疗智能体在诊断—治疗全流程上比肩内科医生；与此同时 AlphaFold 之父 John Jumper 出走谷歌 DeepMind 加入 Anthropic，AI 顶级人才迁徙再起波澜。

## 🎯 Agent 重点

### 1. MIRA 与 AMIE 登上《自然》——首批端到端医疗 AI 智能体表现追平内科医生
- **要点**：自主医疗 Agent 首次在"诊断—治疗"全流程上展现可与人类医生比拼的能力。
- **详情**：最新一期《自然》同期发表两篇论文，分别介绍德国海德堡/德累斯顿团队的 **MIRA（Medical Intelligence for Reasoning and Action）** 与 **Google DeepMind 的 AMIE**。MIRA 在沙箱化 EHR 环境中可自主调用工具完成大量临床任务，AMIE 则采用对话式诊疗范式，两者在多阶段患者管理（问诊、诊断、治疗决策）评测上均"至少与人类内科医生相当"。这是 Nature 首次同期刊载两个独立医疗 Agent 框架，象征"诊断 AI"正向"治理型 AI"切换。
- **影响**：意味着大模型 Agent 在高风险的临床决策链上具备工程可行性，将加速医院 IT、保险核保、远程问诊场景的 Agent 化重构；监管层面对临床 Agent 的可解释性、责任边界讨论也将提速。
- 🔗 [Nature 原论文：Towards autonomous medical artificial intelligence agents](https://www.nature.com/articles/s41586-026-10675-5)
- 🔗 [科技日报/人民网中文报道](https://health.people.com.cn/n1/2026/0622/c14739-40744575.html)

### 2. AlphaFold 之父 John Jumper 离开谷歌 DeepMind，加盟 Anthropic
- **要点**：2024 年诺贝尔化学奖得主、AlphaFold 核心架构师从效力近 9 年的谷歌转投 Anthropic，AI 顶尖人才再次倒向"小而锐"实验室。
- **详情**：6 月 20 日 John Jumper 在 X 平台亲自官宣离职谷歌 DeepMind 副总裁职位，加入 Anthropic 担任研究高层。Jumper 是 AlphaFold 2/3 的灵魂人物，本次跳槽距 Anthropic 估值首次反超 OpenAI（约 9650 亿美元 vs. 8520 亿美元）不到两周。据报道，谷歌 48 小时内连失两位核心 AI 科学家，硅谷一线人才流动进入新一轮高峰。
- **影响**：Anthropic 在科学 / 基础研究方向的话语权显著增强，与其押注的 Claude 长程推理与"AI for Science"叙事高度契合；谷歌内部 AGI 与基础科学路线图压力陡增。
- 🔗 [凤凰科技：AlphaFold 核心缔造者告别谷歌投奔 Anthropic](https://tech.ifeng.com/c/8u6VBTUokvT)
- 🔗 [创业邦：诺奖得主官宣加入 Anthropic 全文](https://www.cyzone.cn/article/837418.html)

### 3. 支付宝"阿宝"内测：超级 App 正式迈向超级 Agent
- **要点**：蚂蚁集团内测 AI 版支付宝，10 亿级用户的支付入口首次被 Agent 重构。
- **详情**：6 月中旬蚂蚁集团内测 AI 支付宝，新增 **"阿宝"** Agent 助手，主界面"往右一滑"即进入对话框，可调度打车、点咖啡、订外卖、查账单、买基金理财等"上万种"小程序能力。这是中国互联网最高频的支付入口首次以 Agent 形态对外开放；与之前后脚，微信也曝出正在内测自家 AI 助手，两大超级 App 几乎同步打响"AI 支付入口"争夺战。
- **影响**：本质是把流量分发从"用户点小程序"变成"Agent 调小程序"，对独立 SaaS / 工具类小程序构成代际冲击；MCP / 小程序协议层将成为下一阶段平台之争。
- 🔗 [腾讯新闻：支付宝和微信这一步，将是 AI 的"扫码时刻"](https://news.qq.com/rain/a/20260618A03Q9B00)
- 🔗 [新浪科技：AI 版支付宝内测界面曝光 搭载"阿宝"助手](https://finance.sina.com.cn/tech/digi/2026-06-15/doc-inicnuhn9923801.shtml)

### 4. OpenAI 据报最快下周推出 GPT-5.6（mini / 标准 / Pro 三档）
- **要点**：紧逼 Anthropic 估值反超，OpenAI 把下一代 GPT 发布窗口锁定 6 月 22–28 日。
- **详情**：格隆汇与多家媒体援引知情人士称，OpenAI 计划在 **6 月 22 日至 28 日**之间发布新一代旗舰 **GPT-5.6**，涵盖 mini / 标准 / Pro 三档。先前 Codex 后台已出现代号 **iris-alpha（鸢尾花）** 的内部模型，疑似为 GPT-5.6 雏形；首席科学家 Jakub Pachocki 告诉员工"GPT-5.6 将比 GPT-5.5 有显著改进"。同期 OpenAI 还在考虑大幅下调 token 定价，以对抗 Claude Fable 5 在编程榜单上的领先。
- **影响**：若如期发布，将直接冲击 Claude Fable 5 编程优势与 Gemini 3.5 Pro 推理优势；价格下调会进一步压缩开源模型推理服务的商业空间。
