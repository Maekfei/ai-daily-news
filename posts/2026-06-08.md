# 🤖 今日 AI 要闻 · 2026-06-08

> 苹果 WWDC 2026 今晚开幕，Siri 2.0 与 Apple Intelligence 全面 AI 重构成最大看点；Supabase 5 亿美元 F 轮融资将估值推至 105 亿美元，AI 后端赛道再起涟漪。

## 🎯 Agent 重点

### 1. 苹果 WWDC 2026 今夜开幕，Siri 2.0 押注 AI 大重构
- **要点**：苹果将首次系统性补齐生成式 AI 短板，Siri 重构与第三方模型开放成核心议题。
- **详情**：北京时间 6 月 9 日凌晨 1 点，苹果（Apple）于库比蒂诺 Apple Park 召开 WWDC 2026。彭博等信源指向代号 "Project Campos" 的 Siri 2.0、定制版 Gemini 1.2T 模型整合，以及 iOS 27 / iPadOS 27 / macOS 27 等六大系统全面 AI 重构。开发者 API 层面预计开放端侧基础模型供第三方调用，并就 Apple Intelligence 在去年跳票的多项功能给出明确路线图。
- **影响**：苹果将正式从"谨慎观望"切换到 Agent 与端侧 AI 全面落地，对 OpenAI、Google、Anthropic 在消费侧的分发格局形成正面挑战。
- 🔗 [腾讯新闻：WWDC 2026 前瞻 — Siri 迎来 15 年来最大变革](https://news.qq.com/rain/a/20260519A03MJL00)

### 2. 微软 Build 2026：发布 7 款自研模型，宣告 "Agent 优先" 时代
- **要点**：微软首次拿出未经其他大模型蒸馏的自研推理模型，并发布开源 Agent 行为控制规范 ACS。
- **详情**：6 月 2 日，微软（Microsoft）在旧金山 Build 2026 开发者大会上一口气发布 7 款 MAI 系列模型，包括首款代码生成模型 MAI-Code-1-Flash 和首个自研推理模型 MAI-Thinking-1。同步开源的 Agent Control Specification（ACS）允许企业以策略文件方式声明智能体可执行/禁止的动作、人工审批节点与审计证据要求，并已支持 LangChain、OpenAI Agents SDK、Anthropic Agents SDK 三大主流 Agent 框架。
- **影响**：微软 AI 底层从依赖 OpenAI 走向自主，企业级 Agent 安全治理首次有了跨厂商的开源开放标准。
- 🔗 [腾讯新闻：一文读懂微软 Build 2026 开发者大会 — "Agent 优先"时代到来](https://news.qq.com/rain/a/20260603A01OUL00)

### 3. 腾讯云数据库面向 Agent 全面升级，"三件套" 对标 Snowflake Cortex
- **要点**：国内首个完整面向 Agent 的数据库矩阵落地，从存储到记忆到分支三件套并发。
- **详情**：5 月 29 日腾讯云发布 DatabaseClaw（Agent 数据闭环）、AgentMemory（长期记忆原生入库）、TDSQL-C Database Branch（数据库分支克隆）三款产品。三件套时间点上紧接 Snowflake Summit 26（6/1-4 旧金山，Cortex Agents + Iceberg V3 全面 GA）与微软 Cosmos Conf 2026（6/5，Agent 长期记忆原生落库 GA），构成全球云数据库厂在 2026 H1 把 "Agent 原生" 做成数据库一等公民的同步动作。
- **影响**：Agent 长期记忆与状态管理首次进入主流数据库厂商的核心产品线，企业落地 Agent 的数据基础设施开始成型。
- 🔗 [新华网：高通 CEO 安蒙 COMPUTEX 2026 演讲 — 智能体已成现实（行业语境）](https://www.xinhuanet.com/tech/20260603/44dd8c3e5fa74da08edc24cf410da524/c.html)

### 4. 高通 CEO 安蒙定义 "2026 是智能体之年"
- **要点**：芯片巨头集体把 Agent 抬升到产业主轴，从 "卖芯片" 转向 "卖算力连续体"。
- **详情**：6 月 1 日 COMPUTEX 2026 开幕主题演讲中，高通（Qualcomm）总裁兼 CEO 安蒙发表《智能体之年》主旨演讲，明确把智能体定义为 AI Token 需求的核心来源，并提出"计算连续体"概念，覆盖云端到边缘终端的全链路 AI 算力布局。同周黄仁勋亦在 GTC Taipei 宣告 Vera Rubin 平台量产、并称 "AI Agent 时代已全面到来"，鸿海、广达、纬创、和硕四大代工厂罕见同台讨论 AI Agent。
- **影响**：硬件叙事从"通用算力"转向"为 Agent 优化的算力"，端侧 NPU、长上下文内存、推理芯片定制化将成为下一轮硬件迭代关键词。
- 🔗 [IT之家：COMPUTEX 2026 — 高通 CEO 安蒙定义智能体之年](https://www.ithome.com/0/958/693.htm)

### 5. 香港发布首个 "生产力级超级智能体"，单次稳定运行 28 小时
- **要点**：HKGAI V3 与配套智能体平台同步发布，主打长时无干预运行。
- **详情**：6 月 3 日，香港生成式人工智能研发中心（HKGAI）发布最新 HKGAI V3 大模型，并推出香港首个生产力级超级智能体平台。官方测试中，该平台可单次无干预稳定运作长达 28 小时，覆盖资料整理、推理分析、报告撰写、代码开发等完整工作流。HKGAI 同步提出 "Token 出海" 战略，希望以本地大模型支撑香港乃至大湾区企业的 AI 出海。
- **影响**：长时长 Agent 任务的稳定性正在从实验室指标走向商用门槛，"无人值守"工作流首次有了可对标的公开数据。
- 🔗 [人民网：香港推出首个生产力级超级智能体](https://hm.people.com.cn/n1/2026/0604/c42272-40733927.html)
