- **要点**：编程 Agent 龙头 Cursor 被马斯克 SpaceX 全资吞并，去年 4 月签署的"600 亿收购或 100 亿合作"二选一期权于 6 月 16 日落地为前者。
- **详情**：SpaceX 通过 SEC 8-K 公告披露交易，Cursor 将作为子公司 X67 Inc. 继续运营，2026 年 ARR 已达 **40 亿美元**。交易为全股票形式，未来 Cursor 团队将与 xAI 的 Grok Code 系列模型深度整合，并接入 SpaceX 内部工程工作流。Cursor CEO Michael Truell 留任，团队约 **300 人** 全员保留。
- **影响**：编程 Agent 三强（Cursor / Claude Code / Codex）格局生变——Cursor 进入"埃隆系"后将获得 xAI 的 Colossus 集群算力补贴，对 Anthropic Claude Code 与 OpenAI Codex 形成更强算力压制；同时也意味着马斯克在生产力 Agent 入口上完成关键卡位。

### 2. Agentjacking 攻击横扫主流编程 Agent，2,388 家组织 Sentry DSN 暴露
- **要点**：Tenet Security Threat Labs 公开新型攻击 "Agentjacking"，仅靠伪造 Sentry 错误日志即可让 Claude Code、Cursor、Codex 自动执行恶意代码，实测成功率 **85%**。
- **详情**：攻击利用 Sentry MCP 集成中开放的 DSN（Data Source Name）注入恶意 issue，AI 编程 Agent 在"自动修 bug"流程里把伪造日志当作可信上下文，调用 shell 工具执行任意命令。研究覆盖 **2,388 家** 公开暴露 DSN 的组织，无需窃取凭据即可触发。Cloud Security Alliance 已于 6 月 12 日发布研究简报，The Hacker News、Infosecurity Magazine 等同步报道。
- **影响**：这是首例公开披露的针对 MCP 协议外部数据源的"间接 Prompt Injection"工业级攻击，倒逼 Anthropic / OpenAI / Cursor 重新评估 Agent 默认信任边界，预计将推动"工具调用沙箱 + 域白名单"成为编程 Agent 标配。
- 🔗 [The Hacker News：Agentjacking Attack Tricks AI Coding Agents Into Running Malicious Code](https://thehackernews.com/2026/06/agentjacking-attack-tricks-ai-coding.html)
- 🔗 [Cloud Security Alliance 研究简报 PDF（Tenet Security 原文）](https://labs.cloudsecurityalliance.org/research/csa-research-note-agentjacking-mcp-sentry-injection-20260612/)

### 3. 微软 Work IQ API 正式 GA，企业 Agent 接入"组织记忆"
- **要点**：Microsoft Build 2026 上预告的 Work IQ API 于 **6 月 16 日正式 GA**，开放给所有 Microsoft 365 Copilot 企业租户与第三方 Agent 开发者。
- **详情**：Work IQ 是微软 IQ 平台（含 Foundry IQ、Fabric IQ、Web IQ）的"工作上下文层"，提供员工关系图、邮件/会议/Teams 频道语义索引、敏感度标签传递三类 API。开发者可让自建 Agent 在保持 Microsoft Purview 合规策略下查询企业级知识，避免重新搭建 RAG。同期上线的 KPMG-Microsoft 联合方案声称将在 100+ 国部署 **5 万** Agent 实例服务审计与税务流程。
- **影响**：对照 Salesforce Agentforce 与 Google Agentspace，微软通过把"组织私域知识"做成 API 抢占企业 Agent 上下文层，是企业级 Agent 落地的关键基础设施动作。
