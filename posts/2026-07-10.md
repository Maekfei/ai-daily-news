## 🎯 Agent 重点

### 1. OpenAI 完成 GPT-5.6 全球全量开放，ChatGPT Work 与 Codex 智能体合并
- **要点**：Sol/Terra/Luna 三档模型全部解锁，编码工具与工作型 Agent 首次进入同一个客户端。
- **详情**：北京时间 7 月 10 日凌晨，**OpenAI** 官宣 **GPT-5.6 系列**（旗舰 **Sol**、均衡 **Terra**、轻量 **Luna**）完成美国政府安全审查后**全球全面开放**，此前自 6 月底起仅对少量合作伙伴限量预览。旗舰 Sol 支持 **Max** 推理强度与 **Ultra** 子智能体并行模式，官方给出的 Terminal-Bench 2.1 得分 **91.9%**（Ultra 模式）超过 Claude Mythos 5。同步上线的 **ChatGPT Work** 是基于 GPT-5.6 的原生工作智能体，能够跨浏览器、文档、企业 SaaS 系统执行长任务；独立 **Codex** 编码工具此次直接并入统一的 ChatGPT 桌面客户端，覆盖 Web / 移动 / Windows / macOS 全终端。
- **影响**：这是自 GPT-4 时代以来 OpenAI 消费端和开发者线最大的一次功能"合并"，把"聊天—编程—工作 Agent"变成同一入口，直接对标 Anthropic 的 Claude Code + Claude Platform 组合，并且给国内厂商（阶跃、豆包、通义、Kimi）出海留出的窗口进一步收窄。

### 2. 阶跃星辰宣布 7 月 13 日发布全球首款「AI 智能体手机」，代工方为华勤技术
- **要点**：大模型公司抢在 OpenAI（计划 2027 年推硬件）之前把 Agent 塞进手机，硬件层 Agent 竞赛正式开局。
- **详情**：7 月 9 日，阶跃星辰（StepFun）官方公众号宣布将推出**新一代智能体终端**，多家一手信源（每经、格隆汇、蓝鲸新闻）证实 7 月 13 日发布**全球首款 AI 智能体手机**，由 A 股上市公司**华勤技术（603296.SH）**代工，双方为"深度绑定合作关系"而非普通贴牌。华勤是全球最大 ODM 之一，其智能手机、笔记本、平板出货量合计**超过全球 10%**，且早在阶跃星辰今年初的 B+ 轮融资中就已入股。阶跃董事长**印奇**多次强调"AGI 必须能与物理世界交互"，公司自 2025 年起即将端侧 Agent 定为核心战略，官方披露**国内 60% 的头部手机品牌**已与其达成合作。
- **影响**：这是首次由国内大模型原厂主导发布的"AI Agent 原生手机"，也是 Agent 从 SaaS 层向硬件层的关键跃迁。若产品体验达标，将迫使苹果 Apple Intelligence、Google Pixel + Gemini Spark、以及国内小米/华为/OPPO 加速自家 Agent 手机路线，硬件+模型+分发的三重整合门槛也将重塑手机行业竞争维度。

### 3. xAI 首个编程智能体模型 Grok 4.5 发布，联合 Cursor 训练
- **要点**：SpaceXAI 上市后的第一张王牌，1.5 万亿参数 V9 底座 + Cursor 真实开发者数据，直指 Claude Opus。
- **详情**：7 月 9 日，马斯克旗下 **SpaceXAI（xAI）** 正式发布 **Grok 4.5**——公司**首个专门针对编程与智能体任务训练**的模型。该模型基于 **V9 底座、1.5 万亿参数**，并引入 AI 编程平台 **Cursor**（SpaceX 数周前刚同意收购）的**数万亿真实开发者编程交互数据**进行补充训练。官方基准：**SWE-bench Pro 64.7%**（超越 GPT-5.5 的 58.6%）、**Terminal-Bench 2.1 83.3%**（与 GPT-5.5 相差 0.1 分）。马斯克将其定位为"Opus 级别"模型，但强调其在速度、Token 效率与成本上更具优势，主打长上下文的大型代码库、多仓库跨技能任务、以及法律和金融专业场景。
- **影响**：xAI 首次拿出一款与 OpenAI/Anthropic 正面对标的编程 Agent 模型，且是"模型公司收购工具公司再共训"的新范式；短期内 Cursor 生态可能出现"Grok 优先"迹象，Claude Code 在编程 Agent 市场的护城河被明显削弱。

### 4. SemiAnalysis 曝 Anthropic Q3 GAAP 利润破 10 亿美元，已秘密递交 IPO 申请
- **要点**：ARR 从 90 亿飙升至 300 亿美元，Anthropic 抢在 OpenAI 前面冲击 AI 实验室最大规模 IPO。
- **详情**：**SemiAnalysis** 7 月 8 日发布的深度报告披露，**Anthropic** 2026 年**三季度 GAAP 利润预计超过 10 亿美元**，对应约 **6% 利润率**；同时其**年度经常性收入（ARR）从 2025 年底的 90 亿美元飙升至 2026 年 Q1 的约 300 亿美元**，主要由 **Claude Code** 驱动。报告确认 Anthropic 已于 **2026 年 6 月 1 日秘密提交 IPO 申请**，若成功上市将成为迄今**规模最大的 AI 实验室 IPO**。SemiAnalysis 引入的"训练前息税利润（EBTIT）"口径显示 Anthropic 二季度 EBTIT 利润率达 36%，并预测 2028 年前累计 EBTIT 将大幅领先 OpenAI。
- **影响**：一年前还被视为"研究驱动、烧钱严重"的 Anthropic，凭借 Claude Code 单一产品线做到了盈利与超高利润率，验证了"Agent 型产品 > 通用聊天订阅"的商业模型；这也把 OpenAI 从"上市节奏领跑者"位置挤下，倒逼其加快自身 IPO 时间表。
