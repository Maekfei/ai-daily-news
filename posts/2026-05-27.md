# 🤖 今日 AI 要闻 · 2026-05-27

> 支付宝公布**3 亿笔智能体支付**并发布 TokenPay/AI 钱包，把 Agent Pay 做成新支付基础设施；OpenRouter 完成 1.13 亿美元 B 轮、估值越过 10 亿美元，"模型路由"上升为 AI 第二层基础设施。

## 🎯 Agent 重点

### 1. 支付宝发布 AI 原生支付体系，3 亿笔智能体支付落地
- **要点**：蚂蚁集团把"Agentic Payment"从概念推到规模化交易，行业首条工业化通路成型。
- **详情**：5 月 27 日上海，支付宝在 Inclusion·外滩大会前哨场宣布累计完成 **3 亿笔 AI 智能体支付**，并一次性发布四款新产品——**AI 付（用户授权智能体代付）、AI 收（商户接入 Agent 流量）、TokenPay（基于支付凭证 token 化的 AI-Ready Card）以及 AI 钱包**——同时支持 95% 主流智能体框架（LangGraph、CrewAI、AutoGen、阿里灵犀 Agent OS、字节 Coze 等）。MiniMax 与阶跃星辰首批接入，将 Agent 端的"挑商品-比价-下单-退款"全链路接到支付宝清算网络。该体系参考国家网信办等三部委 5 月联合印发的《智能体规范应用与创新发展实施意见》，引入"消费者授权-智能体身份-支付凭证"三方校验。
- **影响**：让 Agent 真正具备"完成事情"的最后一公里。Visa、万事达、Adyen 此前与 OpenAI/Google 谈的 AI-Ready Cards 仍在长期机遇阶段，而国内已用规模交易把标准事实化，对跨境支付与 Agent 商业化路径都将产生外溢压力。
- 🔗 [腾讯云开发者社区：智能体自主交易的探索之路](https://cloud.tencent.com/developer/article/2671244)

### 2. OpenRouter 完成 1.13 亿美元 B 轮，估值跃过 10 亿美元
- **要点**：模型聚合层从"工具站"升级为 AI 基础设施第二层，每周吞吐 **25 万亿 token**。
- **详情**：B 轮由 a16z 与 Menlo Ventures 联合领投，Sequoia、SV Angel 跟投。OpenRouter 通过统一 API 接入 OpenAI、Anthropic、Google、xAI、Mistral、Qwen、DeepSeek、智谱、Kimi 等 50+ 模型提供商，按延迟、价格、可用性自动路由；公司披露**周 token 处理量 25 万亿，月活开发者 90 万**。资金将用于建设全球边缘缓存与"agentic routing"——根据任务画像把请求路由到最匹配的模型（推理 / 工具调用 / 长上下文）。
- **影响**：当训练侧的边际收益持续递减，**调度、路由、缓存、计量**这一层的价值被资本明确定价。"OpenRouter 化"也意味着上游模型商对 token 单价的议价能力被削弱，Together、Fireworks、Replicate、硅基流动等同位竞争者将面临估值重定。
- 🔗 [OpenRouter 官方融资公告](https://openrouter.ai/blog/series-b)

### 3. 谷歌 I/O 2026 把"Agent 工具栈"摆到 C 位：Gemini 3.5 Flash + Antigravity 2.0
- **要点**：DeepMind 拿出 **1141 token/s** 的 3.5 Flash 与全新 Agent IDE，主旋律不是"模型更强"，而是"开发者基础设施"。
- **详情**：5 月 26 日凌晨 Mountain View 主会场，**Gemini 3.5 Flash** 以约**前沿模型 4 倍**的生成速度发布，针对长 horizon Agent 任务做了显存与 KV 缓存优化；**Gemini Omni** 引入"世界模型"做对话式视频生成；**Gemini Spark** 是新一代轻量端侧模型；配套的 **Antigravity 2.0** 是面向 Agent 的开发环境，整合 ADK（Agent Development Kit）、A2A 与 MCP 协议、本地沙箱与可视化轨迹回放。Google 同步开放 Gemini Code Assist Agent 给企业版用户，免费配额由 1k 提到 10k req/天。
- **影响**：把过去由 LangGraph、CrewAI、Cursor、Cognition 们零散探索的 Agent 工程问题，第一次官方化、文档化、SDK 化，对独立框架方既是参考也是挤压。
- 🔗 [TechCrunch：With Gemini 3.5 Flash, Google bets next AI wave on agents](https://techcrunch.com/2026/05/19/with-gemini-3-5-flash-google-bets-its-next-ai-wave-on-agents-not-chatbots/)

### 4. Anthropic 准备公开上线最强模型 Claude Mythos 预览版
- **要点**：原本仅对 Glasswing 红队项目开放的 **Mythos**，被发现已在 Claude Code 短暂现身，公开预览或近在数周内。
- **详情**：5 月 26 日 IT 之家与凤凰网科技援引多名内部测试者反馈，Anthropic 正在 Claude Code CLI 后端短暂切换到代号 **"Mythos Preview"** 的新模型，对比 Opus 4.7 在 SWE-bench Verified 与漏洞挖掘任务上**整体提升 20–35%**。Mythos 此前以 244 页 System Card 揭示其"Frontier Red Team"等级——可端到端发现并利用真实开源仓库 N-day 漏洞。Anthropic 已在企业 NDA 客户中开始小范围内测，定价高于 Opus 4.7 的 1.5–2×；不过官方尚未给出准确公开日。
- **影响**：如果 Mythos 公开发布，将与 OpenAI 据传下月发布的 GPT-5.6（150 万 token 上下文）、xAI Grok V9-Medium 形成 6 月模型大战；对国内防御方与安全厂商来说，"闭源前沿模型 + 闭源 Agent harness"的组合压力显著加大。
- 🔗 [IT之家：Anthropic 似准备公开上线 Claude Mythos 预览版](https://www.ithome.com/0/865/521.htm)
