## 🎯 Agent 重点

### 1. Anthropic 让 Claude Artifacts「多人化」，Agent 首次进入实时协同白板
- **要点**：**Anthropic** 将 Claude Artifacts 升级为**多人实时编辑**能力，团队成员可在同一 Artifact 内并行改写、共享 AI 工作区。
- **详情**：7 月 13-14 日，Anthropic 面向 **Claude Team / Enterprise** 计划推出 Artifacts 多人协作模式：多个用户可在同一份由 Claude 生成的**交互式 HTML / React / Notebook Artifact** 内实时改动、指令 Claude 修改，取代此前「导出链接互相传版本」的低效工作流。这是 6 月 18 日 Beta 版 Claude Code Artifacts（含独立 URL 与自动部署）后的关键升级，也是 7 月 7-13 日 **Claude Cowork 扩展到移动端与 Web** 之后 Anthropic Agent 产品线的第三次工程化推进。Anthropic 官方博客同时披露：截至 6 月底，Claude 平台已生成**超过 5 亿个 Artifacts**。
- **影响**：Artifacts 从「AI 生成的单机页面」进化为「AI + 团队实时共同编辑的白板」，这是 Cursor / Replit 之外，第一个原生把「多人 × Agent」放进主流生产力工具的产品设计。对企业 Slack / Notion / Figma 三条线的挤压效应尤为直接。

### 2. OpenAI 发布 ChatGPT Work：跨邮件/Slack/日历的云端 Agent，正式对标 Copilot 与 Cowork
- **要点**：**OpenAI**（Sam Altman）推出 **ChatGPT Work**，一款 **GPT-5.6 驱动**、常驻云端的企业 Agent，可直接接管邮件、Slack、日历、GitHub 中的多步任务。
- **详情**：ChatGPT Work 于 **7 月 10 日**（美东）正式发布，是继 2025 年 1 月 **Operator** 之后 OpenAI 面向企业的第二代 Agent。用户只需输入终极目标（例如「本周把这条产品线的销售汇报和结论发给 CFO」），Work 便可**自主拆解任务**、检索连接的工作工具上下文、跨应用调用 API，并交付**文档 / 表格 / 幻灯**成品。产品与 **ChatGPT Enterprise / Business / Edu** 深度绑定，被媒体解读为 OpenAI **IPO 前**最重要的一次企业化押注。据 VentureBeat，Work 全线基于 **GPT-5.6 Sol** 主模型 + Codex 编排层，Beta 期免费面向 ChatGPT Business 客户，正式版按席位计价。
- **影响**：ChatGPT Work 将 Agent 从「聊天 + tool call」进化到「常驻 SaaS 工位」，直接对撞 Microsoft Copilot 365、Claude Cowork 与 Gemini Enterprise Agent Platform 三家的核心场景。全球「Agent 即员工」的产品叙事进入四家共同定义的阶段。

### 3. 腾讯牵头 20 亿美元回购 Manus：北京把 Agent 公司列为「主权资产」
- **要点**：**腾讯**（Tencent）联合真格基金、HSG 及 Manus 管理层，以**同价 20 亿美元**从 **Meta** 手中回购**新加坡 AI Agent 公司 Manus**；北京要求推翻这笔 2025 年底完成的跨境交易。
- **详情**：据 **英国《金融时报》** 与 **techstartups.com** 7 月 10-11 日披露：Meta 于 2025 年 12 月以 **20 亿美元**收购从中国迁至新加坡的 Manus 后，中国监管机构以「AI Agent 公司属主权级别数据资产」为由要求解除交易；本次买方财团由**腾讯领投**，将成为**最大股东**，真格基金、HSG（原红杉中国）等原有投资方按比例回补。TechTimes 分析指出，该案确立了「**中国血统的 Agent 公司无论注册地在哪、都不得被外资并购**」的新规则；Manus 全球 4000 万累计用户中 60% 来自中国大陆。
- **影响**：这是全球首宗被明确以「Agent 主权化」为由推翻的跨境并购，深刻改变了中美 AI 公司资本流动格局。所有中国血统的头部 Agent 公司（智谱、DeepSeek、月之暗面、Manus 等）事实上被划入「不可跨境出售」清单。
