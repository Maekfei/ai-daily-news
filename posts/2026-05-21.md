## 🎯 Agent 重点

### 1. 阿里云正式提出 "Agentic Cloud"，发布真武 M890 与 128 卡超节点
- **要点**：阿里把"为 Agent 推理而生"写进硬件，平头哥真武 M890 配套 128 卡超节点服务器，目标支撑海量 Agent 并发。
- **详情**：5 月 20 日，2026 阿里云杭州峰会以"Agentic Cloud"为主题，平头哥发布新一代训推一体 AI 芯片**真武 M890**——144GB 显存、卡间带宽 800GB/s，性能为前代真武 810E 的 **3 倍**；同步首发**磐久 AL128 超节点服务器**，单节点搭载 **128 张** M890，配自研互联芯片 **ICN Switch 1.0**，端到端通信时延降至**百纳秒级**。阿里同时官宣 AI 投入"远超 3800 亿元"，已自研 56 万片真武芯片。
- **影响**：这是国内首个把芯片 + 互联 + 整机 + 云调度全栈打包给 Agent 推理的方案，对标 NVIDIA NVL72/GB300 系统，意味着国产算力开始进入"Agent 原生"形态，并进一步拉开与单卡推理路线的差距。

- 🔗 [腾讯网：百纳秒超低时延，阿里真武 M890 128 卡超节点服务器畅跑大模型](https://news.qq.com/rain/a/20260520A04H2N00)

### 2. Andrej Karpathy 加盟 Anthropic，组队用 Claude 加速预训练研究
- **要点**：OpenAI 联创、前特斯拉 AI 负责人 Karpathy 离开个人项目重返大厂，加入 Anthropic 牵头预训练效率研究。
- **详情**：5 月 19 日，Anthropic 发言人确认 Andrej Karpathy 已正式加入，将组建一个新团队，**用 Claude 自身**作为研究伙伴和代码代理，去加速下一代基础模型的预训练流程。Karpathy 此前以 nanoGPT、LLM101n 等开源项目和"vibe coding"概念闻名，近一年密集对外演讲讨论"Agent + 教育"方向。Anthropic 近期密集补强：5 月内既花 3 亿+ 美元收购 Stainless 拿下 SDK 工具链，又官宣 SpaceX Colossus 1 数据中心 22 万张 GPU 算力配额。
- **影响**：在 OpenAI 与 Anthropic 估值反超（Anthropic 1.2 万亿 vs OpenAI）的关键节点，Karpathy 这次站队是顶级 AI 人才流向的标志性信号；他主导的"用 Agent 训练下一代模型"路线，也可能把 Anthropic 的 RLAIF + agentic data 飞轮再加速一档。
- 🔗 [腾讯 KM Media Monitoring May 20：Karpathy 加盟 Anthropic](https://km.woa.com/articles/show/660861)

### 3. Google I/O 2026 合体亚马逊、Meta、微软推出"AI 通用购物车"
- **要点**：谷歌牵头四家巨头共建跨平台 AI 电商协议，让 Agent 在 AI 搜索结果里直接完成支付。
- **详情**：5 月 20 日 Google I/O 2026 上，谷歌联合亚马逊、Meta、微软发布"AI 通用购物车"协议草案，Agent 可以在 Gemini App、AI Mode 搜索结果、Chrome Agent 中调起统一支付与配送接口；同时 Pichai 透露 **Gemini App 月活已突破 9 亿**（一年翻倍），AI Mode 搜索默认开启后 **48% 的搜索**触发 AI Overviews，零点击率升至 58.5%。
- **影响**：此前 Stripe Sessions 2026 已宣布 Stripe 与 Google AI Mode、Gemini App 打通；本次四巨头协议把"Agent 商务"标准化，等于以电商为切口推动 Big Tech 共建 Agent 互操作层，挤压垂类导购代理的存在空间。
