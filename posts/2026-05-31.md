## 🎯 Agent 重点

### 1. Anthropic 估值反超 OpenAI，Claude 在企业端跑通商业闭环
- **要点**：Anthropic 二级市场估值越过 1 万亿美元，企业客户数与 ARR 同步反超 OpenAI。
- **详情**：路透与 CNBC 援引知情人士消息，Anthropic 正在敲定新一轮融资，估值区间 1.0–1.2 万亿美元，Claude 企业版年化收入（ARR）超过 90 亿美元，主要由 Claude Code 在编程场景的结构性替代驱动；同期 OpenAI 已秘密向 SEC 提交 IPO 招股说明书，目标估值同样为 1 万亿，由高盛和摩根士丹利联合承销，最早 9 月上市。两家正面对决从模型层延伸至资本层。
- **影响**：AI 公司估值锚由"用户规模"切换到"企业工作流嵌入深度"，Coding Agent 正成为重新定价的核心变量。
- 🔗 [aibetas：Anthropic 估值反超 OpenAI 全景](https://www.aibetas.com.cn/p/6257.html)

### 2. Anthropic 推出 Claude Opus 4.8 与 Dynamic Workflows，Agent 进入"动态编排"时代
- **要点**：Opus 4.8 配合 Dynamic Workflows，让 Claude 在运行时按需拼装 sub-agent 与工具链。
- **详情**：Anthropic 同步放出 244 页系统卡，Opus 4.8 在 SWE-Bench Verified、Terminal-Bench 2.0 上较 4.7 再涨 4–6 个百分点，并首次官方支持 Dynamic Workflows——开发者只需声明目标，Claude Code 即在执行中自动决定串行/并行 sub-agent、工具调用顺序与回滚策略。配合此前曝光的最强模型 Mythos 1（在 Claude Code 后台短暂现身），Anthropic 三张底牌（Sonnet 4.8 / Opus 4.8 / Mythos 1）已全部翻开。
- **影响**：Agent 编排从"开发者写图"进化到"模型自规划"，对 LangGraph/AutoGen 这类静态 DAG 框架是一次降维冲击。
- 🔗 [Anthropic 官方：Introducing Dynamic Workflows in Claude](https://claude.com/blog/introducing-dynamic-workflows-in-claude)
- 🔗 [新智元：Anthropic 三张底牌全翻——Mythos 1 现身、Opus 4.8 曝光](https://news.qq.com/rain/a/20260524A049FF00)

### 3. OpenAI GPT-5.6 下月发布，150 万 Token 上下文剑指"超级智能体"
- **要点**：开发者从 Codex 后端日志中抓到内部代号 iris-alpha 的新模型，预计 6 月上线。
- **详情**：IT之家、51CTO 报道，多名开发者在 OpenAI Codex 后端配置中发现未官宣模型 GPT-5.6，内部代号 iris-alpha，**支持 150 万 token 上下文窗口**，距 GPT-5.5 发布仅 3 周。Sam Altman 在内部全员会重申 GPT-5.6 将作为"超级智能体（super-assistant）"基座，主打长任务自主执行（multi-day task）与全员可用 ChatGPT Agent。
- **影响**：百万级 token 上下文基本成为头部模型 2026 年下半年标配，Agent 长程记忆/长任务规划的工程瓶颈再次后移。

### 4. 中国信通院联合三大运营商发布 ATH 1.0：全球首个 Agent 可信握手协议
- **要点**：智能体可信握手协议（Agent Trusted Handshake 1.0）开源，专攻 Agent 调 Agent / Agent 调 App 的身份与权限。
- **详情**：信通院联合中国电信、中国联通、中移九天、腾讯（玄武实验室 + CSIG PA）、华为、中兴、港中大（深圳）等机构联合编制，已在 GitHub 与 AtomGit 双源开源。ATH 1.0 定义"用户—调用方 Agent—被调方 Agent/App"三方握手流程，覆盖身份核验、权限粒度授权、审计回溯三大环节，对标 Anthropic 主导的 MCP 但更强调监管侧可审计。同日中国电信宣布"以 Token 经营重塑公司业务"，联通押注 Agent 与 AI 云。
- **影响**：MCP 之外的国产 Agent 互联标准开始落地，运营商正式进场 Agent 基础设施层，万亿级智能体生态的"连接券"竞争开打。
- 🔗 [腾讯 KM：ATH 1.0 协议解读——万亿级智能体生态的连接券](https://km.woa.com/articles/show/660372)

### 5. DeepSeek 联合 CodeAgent 发布《From Copilots to Coworkers》：99% 由 AI 自己写
- **要点**：45 页论文 99% 内容由 AI 执笔，提出 L1–L5 自主研究 Agent 分级，业界主流停留在 L4。
- **详情**：DeepSeek 研究员与 CodeAgent 团队联合署名的最新论文系统梳理了 Coding Agent 自主性等级：L1 补全 → L2 重构 → L3 多文件 → L4 项目级自主交付 → L5 自主立项+科研。论文得出结论，Cursor、Claude Code、Codex CLI 等头部产品均属 L4，距 L5 仍隔"长程规划+实验闭环"两道坎；DeepSeek 同步开源对应评测集 SE-Coworker-Bench，覆盖 12 类真实 SWE 任务。
- **影响**：给业界一把统一尺子去衡量"AI 同事"成熟度，也为后续的 Agent 招标/采购提供分级语言。
- 🔗 [iWiki AI 行业速递（2026-05-29）：DeepSeek 论文与 L1–L5 分级](https://iwiki.woa.com/p/4021047760)
