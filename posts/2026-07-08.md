## 🎯 Agent 重点

### 1. Norm Ai 完成 1.2 亿美元 C 轮，法律 AI 又一独角兽
- **要点**：Khosla Ventures 领投，法律与合规垂直 Agent 赛道再添 12 亿美元估值玩家。
- **详情**：位于纽约的法律与合规 AI 公司 **Norm Ai** 7 月 7 日宣布完成 **1.2 亿美元 C 轮融资**，投后估值 **12 亿美元**，正式跻身独角兽。本轮由 **Khosla Ventures** 领投，Bain Capital Ventures、Coatue、Vanguard 等参投。Norm 的核心产品是把美国 SEC、FINRA 等监管法规转成可执行的"合规 Agent"，供华尔街投行和保险公司实时审查营销素材、员工邮件与理赔文档。相比 Harvey（估值约 50 亿美元）和 Legora，Norm 走的是"深度垂直+合规审计"路线，估值仅为前者五分之一。
- **影响**：法律科技仍是 2026 年 Agent 商业化最跑得通的垂直之一，但估值分层已经出现——通用助手估值高、合规审计估值低但 ARR 更扎实。
- 🔗 [TechCrunch: AI law startup Norm raises $120M](https://techcrunch.com/2026/07/07/ai-law-startup-norm-raises-120m-hits-unicorn-valuation/)
- 🔗 [Norm 官方公告](https://www.norm.ai/resources/norm-ai-raises-20-million-at-a-1-2-billion-valuation)

### 2. Bespoke Labs 获 4000 万美元，专攻"训练可靠 Agent 的强化学习环境"
- **要点**：Wing VC 与 8VC 双领投，标志 Agent 基础设施正在向"训练环境"这一新赛道分化。
- **详情**：位于加州 Mountain View 的 **Bespoke Labs** 7 月 6 日宣布完成合计 **4000 万美元融资**（8VC 领投的种子轮 + Wing VC 领投的 A 轮）。公司由前 Google Research 高级研究员 **Mahesh Sathiamoorthy** 与 **Alex Dimakis 教授** 于 2024 年创立，为前沿实验室和企业构建**强化学习环境（RL environments）与基础设施**，让 Agent 在部署前先在模拟工具、API、代码库和企业系统里学会"可靠地失败与恢复"。融资将用于研究、招聘和商务拓展。
- **影响**：Agent 从"能用"走向"可靠"的最大瓶颈是环境稀缺——公开数据训不出真实企业级 Agent。Bespoke 这轮反映资本正在从"更大模型"转向"更好训练环境"，与 Scale AI、Surge 的路线形成互补而非替代。
- 🔗 [Wedbush Investor Wire: Bespoke Labs Announces $40M](https://investor.wedbush.com/wedbush/article/bizwire-2026-7-6-bespoke-labs-announces-40m-to-build-the-environments-that-train-reliable-agents)
- 🔗 [Let's Data Science: Bespoke Labs Raises $40M for Agent Training](https://letsdatascience.com/news/bespoke-labs-raises-40m-for-agent-training-15deac2e)

### 3. Anaplan 发布 "Agentic Enterprise"，把 Agent 塞进企业决策链
- **要点**：Amazon Bedrock 上部署，主打"决策级 Agent"而非聊天助手。
- **详情**：企业规划巨头 **Anaplan** 7 月 7 日正式发布 **Agentic Enterprise**，一个部署在 **Amazon Bedrock** 上的可信 AI 决策基础设施，把 Anaplan 平台的多维建模能力和 Agent 推理能力打通，让 Agent 能够在预算、供应链、销售规划场景中自主提出方案、评估情景、落实执行。首批应用聚焦 FP&A（财务规划分析）与 S&OP（销售与运营计划）。产品与 AWS 深度绑定，凸显亚马逊在企业级 Agent 分发上的重要布局。
- **影响**：Anaplan 是全球财务规划工具头部厂商，此举意味着 Agent 正在从"生产力工具"下沉为"企业中台一等公民"，直接与 SAP Joule、Salesforce Agentforce、Oracle AI Agent Studio 争夺 CFO 预算。
- 🔗 [Anaplan 官方：Introduces the Agentic Enterprise](https://www.anaplan.com/news/anaplan-introduces-the-agentic-enterprise/)
- 🔗 [AI Agents Directory: July 7 news brief](https://aiagentsdirectory.com/news/ai-agents-news-brief-july-7-2026)

- **要点**：展览面积首破 10 万平方米，形成上海世博/张江/西岸"三地四馆"布局。
- **详情**：**2026 世界人工智能大会暨人工智能全球治理高级别会议** 将于 **7 月 17 日至 20 日**在上海举办，主办方 7 月 7 日召开新闻发布会介绍：本届大会在展览规模、首发新品数量、论坛场次等方面均创下历届新高，展览面积首次突破 **10 万平方米**，上千家参展企业将带来 **3000 余项**展品，其中包含**上百款首发新品**。大会主论坛将同步举办人工智能全球治理高级别会议，联合国机构与多国官员将出席。
