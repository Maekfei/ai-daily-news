## 🎯 Agent 重点

### 1. 斯坦福 Biomni 登《Science》：通用生物医学 AI 智能体自主完成科研任务
- **要点**：斯坦福团队推出 **Biomni**——首个可端到端执行数据分析、实验设计的通用生物医学 Agent，多任务表现「接近人类专家」。
- **详情**：**7 月 12 日**，**斯坦福大学** James Zou / Serafim Batzoglou 团队在《**Science**》发表长文，正式发布 **Biomni**——一款以「AI 共同科学家（co-scientist）」定位的通用生物医学智能体。与聊天机器人不同，Biomni 可将自然语言问题拆分为多步子任务，自主调用**基因组分析、单细胞聚类、湿实验方案生成、文献综述**等专业工具链；在克隆设计、GWAS 分析、罕见病诊断等**多项案例研究**中，其结果被评估为**接近领域专家水平**。项目同期开放官网 biomni.stanford.edu 与开源代码，并已与多家生物制药实验室开展早期合作。
- **影响**：这是首个登顶顶级学术期刊的**通用型科研 Agent**，为「AI Agent + 湿实验」的科研范式提供了同行评审级证据；与前一周 Meituan 在 ACL 上展示的 Self-Evolving Agent 系统一起，标志着 Agent 从「白领效率工具」向「专业科研生产力」跨越的时点已到。

### 2. Prime Intellect 发布 Verifiers v1：Agent RL 训练框架首次「任务/沙盒/环境」三层解耦
- **要点**：Prime Intellect 开源 **verifiers 0.2.0（v1 API）**，把 Agent 强化学习环境拆成 **TaskData / Harness / Runtime** 三层可组合模块，被 AlphaSignal 评为「从底层重写 Agent 训练」的关键版本。
- **详情**：**7 月 10 日**（美东），**Prime Intellect**（Intel Capital 投资的分布式 AI 训练公司）发布博客与 **GitHub Release v0.2.0**，正式推出 **verifiers v1**：核心变化是把 Agent RL 训练环境**拆成三层可组合原语**——**TaskData**（每一行可序列化的题目数据）、**Harness**（复用性执行协议，如命令行 Agent、第三方框架适配器、Sandbox 主机）、**Runtime**（进程与资源）。同时开箱内置 **MCP tools、endpoint 路由、嵌套 harness、生命周期装饰器**等 12 项工程化能力。v1 直接对齐 Prime Intellect 上周披露的「**Self-Improving Agents 全栈训练平台**」愿景——他们把 verifiers 定位为 **Agent 的 datasets + PyTorch**。
- **影响**：Agent RL 是继 pre-training / SFT / RLHF 之后被公认为下一波产能来源，但此前生态里 SWE-Gym、SWE-Rex、AgentBench、τ-Bench 各自为政。verifiers v1 提供了**跨环境统一 IO 契约**，直接对标 Anthropic 内部的 Claude Code training harness，是开源社区第一个可用的对手方案；对复现 GRPO / Agentic RL 论文的团队意义重大。
- 🔗 [Prime Intellect 官方博客：verifiers v1 — Decomposing Tasksets and Harnesses for Agentic RL](https://www.primeintellect.ai/blog/verifiers-v1)
- 🔗 [GitHub Release：PrimeIntellect-ai/verifiers v0.2.0](https://github.com/PrimeIntellect-ai/verifiers/releases)
- 🔗 [AlphaSignal：Prime Intellect's Verifiers v1 Rewrites AI Agent Training From the Ground Up](https://alphasignal.ai/news/prime-intellect-s-verifiers-v1-rewrites-ai-agent-training-from-the-ground-up)

### 3. Meta 开放 Muse Spark 1.1 API：正式加入编程与 Agent 商业化战局
- **要点**：Meta Superintelligence Labs 发布 **Muse Spark 1.1**，同步开放 **Meta Model API 公开预览**，输入价 **$1.25 / 百万 token**，被官方称为「迄今最强的 Agent 与编程模型」。
- **详情**：**7 月 9 日**，**Meta** AI 负责人 **Alexandr Wang** 宣布 Muse Spark 1.1 正式上线并开放 **Meta Model API**——这是 Meta 首次以「按 token 收费」模式对开发者开放前沿模型，被市场视为对齐 Anthropic / OpenAI 商业化路径的关键动作。Muse Spark 1.1 在**工具调用、电脑操作、代码生成、多模态理解**四项 Agent 关键维度做出跨代升级，Meta 表示相比初代「基于开发者反馈重训 harness 与工具协议」。价格上 **$1.25/M 输入**、$5/M 输出 —— 略低于 Claude Sonnet 5，直接卡位「Agent 编程性价比档」。
- **影响**：Meta 从「只做开源、拒绝商业化 API」转向「开源 + 付费 API 双轨」，意味着一线大厂在 Agent 底座上再无「非商业化玩家」；随着 GPT-5.6 Sol、Claude Sonnet 5、Gemini 3.1 Pro、Muse Spark 1.1 齐聚，2026 下半年的编程 Agent 之战将呈现「四强 + Kimi K2/DeepSeek V4 开源双雄」的格局。
