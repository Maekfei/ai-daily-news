## 🎯 Agent 重点

### 1. 阿里整合 QoderWork / 悟空 / MuleRun 三大 Agent 产品线
- **要点**：钉钉新任 CEO 陈宇森"新官上任第一把火"，把阿里三条 Agent 产品线拧成一款企业级 AI。
- **详情**：7 月 2 日，阿里巴巴宣布以 2026 年 1 月推出的桌面 AI 智能体工具 **QoderWork** 为底座，深度整合钉钉孵化的企业协同 Agent"悟空"和阿里云内部创业项目 **MuleRun** 的能力，推出面向企业生产力场景的全新 AI 产品。新产品由 6 月刚接任悟空事业部 CEO、并兼任钉钉 CEO 的 90 后陈宇森操盘。三款现有产品服务将无缝升级，用户权益不受影响。
- **影响**：阿里从"多点开花"转向"一支主力"，与钉钉的企业渠道深度绑定，直接对标微软 Copilot Studio 的企业 Agent 生态；也标志着国内大厂 Agent 竞争从"堆产品"进入"整合收敛"阶段。
- 🔗 [IT之家：阿里拟整合 QoderWork、悟空、MuleRun 三大 Agent 产品](https://www.ithome.com/0/971/554.htm)

### 2. xAI 发布 Voice Agent Builder 测试版：2 分钟搭建生产级语音 Agent
- **要点**：马斯克 xAI 推出零代码语音智能体平台，主打"两分钟上线企业客服"。
- **详情**：7 月 2 日，xAI 发布基于自研 **Grok Voice** 模型的 Voice Agent Builder Beta，端到端集成电话通信、知识库检索、工具调用与转人工能力，声称在 τ-bench 类评测上达到生产级表现。计费模式为按使用时长 **0.05 美元/分钟**，官方演示中"两分钟"即可完成从提示词到可拨打电话号码的完整部署。
- **影响**：语音 Agent 是 2026 年增长最猛的细分赛道之一（CB Insights 六大趋势之首），xAI 直接对标 Retell、Vapi、ElevenLabs Agents，把接入门槛压到接近零；同时 Grok Voice 也第一次以独立商业化产品形态对外，为 xAI 商业收入贡献新支点。
- 🔗 [虎嗅：xAI 发布 Voice Agent Builder 测试版](https://www.huxiu.com/ainews/13655.html)

### 3. 2026 中国 AI 智能体大会杭州开幕：Agent 从概念转入工程化
- **要点**：阿里、腾讯、华为、美团、字节、微软六大厂齐聚杭州，主题聚焦 Agent 工程化落地。
- **详情**：7 月 2 日，2026 中国 AI 智能体大会在杭州开幕。会议核心议题从"Agent 是什么"转向"Agent 怎么规模化跑起来"——包括 MCP/A2A 协议标准化、多 Agent 编排、企业观测与治理、成本控制等硬工程话题。多家厂商在会上分享了各自 Agent 中台的百万级并发实践与 token 成本优化经验。
- **影响**：这场大会被业内视为国内 Agent 从 demo 时代进入"生产系统"时代的分水岭标志。协议层（MCP、A2A）、基础设施层（Agent 中台）、应用层（垂直场景）的分工正在明确，也是钉钉、飞书、腾讯乐享等企业协作平台加速内嵌 Agent 的信号。
- 🔗 [CSDN：中国 AI 智能体大会今日开幕，阿里腾讯华为全到齐](https://blog.csdn.net/smallym1/article/details/162517634)
