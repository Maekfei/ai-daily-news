## 🎯 Agent 重点

### 1. 中国发布《人工智能 智能体互联》系列 7 项国家标准
- **要点**：全球首个智能体互联国家标准体系落地，破解"智能体孤岛"。
- **详情**：6 月 26 日，市场监管总局联合工信部召开新闻发布会，正式发布 GB/Z 185—2026《人工智能 智能体互联》系列 7 项国家标准，覆盖**总体架构、身份码、身份管理、智能体描述、智能体发现、智能体交互、工具调用**等核心环节。中国电子技术标准化研究院牵头制定，针对当前不同厂商 Agent 之间接口/协议不统一形成的"智能体孤岛"，建立统一身份认证和全程追溯机制。
- **影响**：标准实施后企业可复用标准组件压缩上市周期，对国内 MCP/A2A 类协议生态形成对标，为跨域可信交互奠定制度基础，预计将加速国产 Agent 的规模化协同应用。

### 2. General Intuition 完成 3.2 亿美元 A 轮，估值 23 亿美元
- **要点**：用游戏数据训练具身 Agent 的 AI 实验室刷新赛道纪录。
- **详情**：6 月 25 日，纽约/荷兰 AI 实验室 General Intuition 宣布完成 **3.2 亿美元 A 轮融资**，投后估值 **23 亿美元**，累计披露融资达 4.54 亿美元（含 2025 年 10 月的 1.34 亿美元启动轮）。公司用**数百万小时玩家游戏录像**训练世界模型，让 Agent 学会"接近人类直觉"的动作规划，并已演示同一模型同时驱动游戏内角色与现实机器人。
- **影响**：游戏环境产生的动作数据被资本重新估价为训练真实世界 Agent 的稀缺资产，在 Genie 3、Veo 等世界模型热潮中给具身智能赛道注入新的数据范式。

### 3. OpenAI 发布 GPT-5.6 系列：Sol/Terra/Luna 三档登场
- **要点**：旗舰 Sol 在 Terminal-Bench 2.1 智能体编程基准上以 88.8% 反超 Claude Mythos 5（88.0%）。
- **详情**：6 月 26 日 OpenAI 正式发布 GPT-5.6 系列，包括旗舰 **Sol**、均衡版 **Terra**、轻量版 **Luna**。Sol 引入新的 **Max 推理强度**和借助子智能体加速复杂任务的 **Ultra 模式**，开启 Ultra 后 Terminal-Bench 2.1 得分跃升至 **91.9%**。Terra 性能对标 GPT-5.5 但成本降低 2 倍。Sol API 价格为输入 5 美元/M、输出 30 美元/M tokens；该系列首次让所有型号在网络安全/生化两个维度均被评级为 **High Risk**，并因美国政府要求仅向"可信合作伙伴"有限开放。
- **影响**：以子 Agent 调度作为旗舰能力卖点，意味着 Agentic 编程已成主流大厂下一代模型的核心差异化战场；同时美国政府对前沿模型发布的事前审查机制首次在商业产品上落地。

- 🔗 [新浪科技：编程跑分超 Claude Mythos 5](https://finance.sina.com.cn/tech/digi/2026-06-27/doc-inieuyie1636480.shtml)

### 4. 英伟达×剑桥发布"红皇后哥德尔机"：Agent 与评测器协同进化
- **要点**：首个让评测器与被评 Agent 共同进化的自我改进框架，破解评测饱和盲点。
- **详情**：6 月 24 日剑桥大学与 NVIDIA 等 13 位作者联合在 arXiv 提交预印本（编号 **2606.26294**）《Red Queen Gödel Machine》。该 RQGM 框架在自我改进 Agent 之外**额外学习一个评测器**，两者按"红皇后假说"动态博弈共同进化，解决了静态基准下自改进 Agent 一旦饱和就停滞的问题。论文称在多个 agentic coding benchmark 上达到 SOTA，让 AI 可以"无限刷级"地自我繁衍代码。
- **影响**：被业内称为"年度最危险论文"——若可大规模复现，将动摇当前依赖固定 benchmark 的对齐与安全审查范式，也可能催生新一代自演化 Agent 训练流水线。
- 🔗 [arXiv 2606.26294：Red Queen Gödel Machine](https://arxiv.org/abs/2606.26294)
- 🔗 [黑豹商业评论：剑桥×NVIDIA 红皇后哥德尔机解读](https://www.heibaos.com/post/372619164844038/red-queen-godel-machine-cambridge-nvidia-june-2026)
