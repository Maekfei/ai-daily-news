## 🎯 Agent 重点

### 1. 字节火山引擎发布豆包 2.1 Pro，All-in Coding 与 Agent
- **要点**：豆包 Seed 2.1 三大优化方向锁定 Coding、Agent、VLM，配套升级整套 Agent 云服务体系。
- **详情**：6 月 23 日，火山引擎在北京 2026 FORCE 原动力大会发布豆包大模型 2.1（含 Pro / Turbo 双版本）以及视频、图像、音频多款新模型。火山引擎披露豆包大模型日均 Token 使用量已突破 **180 万亿**，相较 2024 年 5 月 1200 亿规模两年间增长 **约 1500 倍**；火山引擎在中国公有云 MaaS 市场份额达 **49.5%** 居首，年消耗超 1 万亿 Token 的企业客户已超 200 家（去年底为 100 家）。官方称豆包 2.1 多项评测优于 Anthropic Claude 同档模型。
- **影响**：标志国内头部厂商从"卖模型"全面转向"卖 Agent 工作流"，Coding 与智能体场景被定义为下一个 Token 消耗主战场。
- 🔗 [腾讯新闻：豆包 2.1 Pro 发布，Coding 与 Agent 跨越质变点](https://news.qq.com/rain/a/20260623A04X5J00)

### 2. OpenAI 与博通联合发布首款自研推理芯片 Jalapeño
- **要点**：OpenAI 9 个月内完成从设计到流片，宣称推理成本可降低约 50%。
- **详情**：北京时间 6 月 24 日，OpenAI 联手博通（Broadcom）正式发布首款定制 AI 推理芯片 **Jalapeño**（哈拉贝诺辣椒）。芯片采用 TSMC **3nm** 工艺，脉动阵列架构搭配 HBM3E/HBM4，由博通提供 Tomahawk 交换芯片处理网络层。OpenAI 披露研发过程中调用自家大模型加速优化流程，工程样片已稳定运行 GPT-5.3-Codex-Spark 等前沿模型，每瓦能效"大幅超越当前市面顶尖产品"。OpenAI 同时披露多代计算平台计划，**2026 年底前完成首次部署**。
- **影响**：继 Google TPU、AWS Trainium 之后，OpenAI 正式加入自研芯片俱乐部，进一步压低对 NVIDIA 的依赖，也为博通的 ASIC 业务再添一笔大单。
- 🔗 [第一财经：OpenAI 发布首款定制 AI 推理芯片，披露多代计算平台计划](https://www.yicai.com/news/103244890.html)
- 🔗 [腾讯新闻：刚刚，OpenAI 交出第一颗自研芯片](https://news.qq.com/rain/a/20260624A0B0Q000)

### 3. 豆包推出付费专业版，可直接操作本地电脑
- **要点**：豆包 2.1 同步推出 C 端付费专业版，赋予智能体执行复杂任务和操作本地电脑的能力。
- **详情**：财新报道，豆包 2.1 在面向企业的 API 之外，C 端首次推出**付费专业版**，可执行连贯智能体任务、调用本地工具与文件系统。这是国内主流大模型 App 首次大规模商业化"Agent 模式"，对标 Claude Code、ChatGPT Agent 等海外产品。
- **影响**：国产智能体产品从纯 Chat 转向"能动手"，用户付费心智从"问答助手"切换到"自动化打工人"。
- 🔗 [财新：豆包推出付费专业版 可执行智能体任务、操作本地电脑](https://www.caixin.com/2026-06-24/102456917.html)

### 4. ISC.AI 2026 大会聚焦 "AI 对 AI" 安全范式
- **要点**：6 月 24 日北京举行的第十四届互联网安全大会主轴明确转向智能体安全。
- **详情**：360 集团周鸿祎在主论坛指出 2026 年是"百模大战进入 Agent 实战年"，安全边界从"人对人"过渡到"AI 对 AI"。大会发布多项 Agent 沙箱与红队评测工具，会上披露智能体规模化部署带来的提示词注入、工具滥用、Agent 间合谋等新型风险。
- **影响**：安全行业首次系统化讨论 Agent 治理，给后续企业级智能体落地划出合规底线。
- 🔗 [新浪财经：ISC.AI 2026 定档 6 月 24 日 聚焦"人对人"向"AI 对 AI"安全范式变革](https://finance.sina.com.cn/roll/2026-06-02/doc-inhzzivq4809310.shtml)
