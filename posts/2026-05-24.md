## 🎯 Agent 重点

### 1. DeepSeek 700 亿元首轮融资接近敲定，宁德时代拟参投，承诺继续开源
- **要点**：DeepSeek（深度求索）首轮融资规模锁定 **700 亿元人民币**，宁德时代拟参与，公司明确仍将以"突破性 AI 研究 + 开源"为长期战略。
- **详情**：5 月 22 日彭博社首发、5 月 23 日宁德时代参投消息进一步流出。DeepSeek 高层向投资者明确表示：融资目标 **700 亿元 RMB**，未来将优先投入"突破性的 AI 研究"，而不是短期商业化变现，模型与权重将持续开源。一位接近交易的人士透露估值在 1500–1800 亿元区间，相比 2025 年底估值上调约 3 倍。配套动作上，DeepSeek 旗舰模型 V4 系列已宣布 API 永久降价，其内部正搭建一个对标 Claude Code / OpenAI Codex 的 Coding Agent harness 团队。
- **影响**：这是国产开源模型阵营 2026 年最大的单笔融资意向，与 OpenAI / Anthropic 的"高估值闭源 + IPO"路径形成正面对照。开源 + 大资本 + 长期主义的组合，对全球 Agent 生态、推理成本曲线（每百万 token 价格已较去年下降 80%+）将产生持续向下压力。

### 2. 智谱发布 GLM-5.1 高速版，400 tokens/s 刷新全球大模型 API 速度上限
- **要点**：智谱 AI（02513.HK）推出 GLM-5.1-highspeed，企业 API 输出速度达 **400 tokens/s**，首次将旗舰级大模型能力与极致低延迟结合，发布当日港股一度涨超 22%。
- **详情**：5 月 22 日，智谱面向部分企业客户开放 GLM-5.1 高速版 API "GLM-5.1-highspeed"。模型基于旗舰 GLM-5.1 蒸馏而来，单 stream 输出速度 **400 tokens/s**，刷新全球大模型 API 速度上限——此前同等速度档位主要集中在 7B–14B 量级的轻量模型。智谱称该版本由 GLM 团队与硬件加速厂商 Tile（瓦片智能）联合优化，30 秒可生成一段复杂网页代码，端到端首 token 延迟 < 100ms。智谱（02513.HK）港股盘中涨幅一度达 **22%**，收盘附近回落至 +20%、报 1219 港元。
- **影响**：高速 + 旗舰能力的组合直接打击 Agent 场景的"延迟焦虑"——多步规划、Tool Use、长 Horizon 编码 Agent 对 token/秒尤其敏感。GLM-5.1-highspeed 把"用得起 + 跑得快"重新拉回到国产开源模型的竞争叙事中。
- 🔗 [腾讯新闻 - 智谱推出 GLM-5.1 高速版：400 tokens/s 30 秒生成复杂网页代码](https://news.qq.com/rain/a/20260522A04T9T00)
- 🔗 [新浪财经 - 智谱(02513) 推出 GLM-5.1 高速版 API 400 tokens/s 刷新全球速度上限](https://finance.sina.com.cn/stock/hkstock/ggscyd/2026-05-22/doc-inhytqkv7800210.shtml)

### 3. Cursor Composer 2.5 上线，基于 Moonshot Kimi K2.5 持续预训练
- **要点**：Cursor 发布全新代码 Agent 模型 **Composer 2.5**，基于 Moonshot AI 的 **Kimi K2.5** 开源 checkpoint 进行持续预训练 + 强化学习训练，定位"规划、构建、调试一体"。
- **详情**：5 月中下旬 Cursor 官方、设计师 Ryo Lu 在 X 平台密集预热后，Composer 2.5 在 5 月 19–22 日正式推送。技术亮点：① 基础底座取自 Moonshot Kimi K2.5 开源权重；② 引入"基于文本反馈的定向 RL"，针对长 rollout（数十万 token）下的信用分配问题做了专门优化；③ 配合 Cursor 3 的 Agent Workspace，可以多个本地/云端 Agent 并行；④ 与 Cursor Design Mode 联动，特别适合 UI/前端任务。社区实测在 SWE-Bench-Verified 与内部代码库迁移任务中，相对 Composer 2 的成功率提升约 18–25%。
- **影响**：这是头部"AI Coding 闭源产品 + 中国开源底座"的标志性合作之一。Cursor 用 Kimi K2.5 + 自研 RL 训练做出旗舰模型，证明开源底座 → 闭源 Agent 上层这一商业模式在工业级编码 Agent 上完全可行；也是 Moonshot 海外品牌影响力的重要信号。
- 🔗 [The New Stack - Cursor releases Composer 2.5 with Kimi K2.5](https://thenewstack.io/cursor-releases-composer-2-5-with-kimi-k2-5/)
- 🔗 [The Decoder - Cursor's new coding model Composer 2.5 is built on Moonshot's Kimi K2.5](https://the-decoder.com/cursor-composer-2-5-kimi-k2-5/)
