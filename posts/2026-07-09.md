## 🎯 Agent 重点

### 1. 工信部 NVDB 通报 Claude Code 存在安全后门，阿里 7 月 10 日全员卸载
- **要点**：中国国家级漏洞平台首次点名头部海外 AI Agent 产品，Agent 供应链安全上升到国家议程。
- **详情**：**工业和信息化部网络安全威胁和漏洞信息共享平台（NVDB）** 7 月 8 日发布风险提示，指 Anthropic 开发的 AI 编程工具 **Claude Code** 内置监控机制，可在未经用户同意的情况下向远程服务器回传**用户地域、身份标识等敏感信息**，"危害严重"。同日阿里巴巴宣布将 Claude Code 列入**高风险软件名单**，包括 Sonnet、Opus、Fable 全系模型和 Claude Code 智能体产品在**7 月 10 日**正式禁用，并推荐 Qoder 作为替代方案。这是国内头部大厂首次对海外 Agent 产品做出全面封禁级动作。
- **影响**：Agent 产品要访问用户完整代码库、终端与本地文件，供应链安全边界远高于普通 SaaS；此事件将加速国内企业转向自研或国产 Agent 工具链，并可能引发欧洲监管跟进。

### 2. 阿里巴巴合并 QoderWork/悟空/MuleRun 三大 Agent 产品，港股飙升 12.2%
- **要点**：以桌面智能体 QoderWork 为底座，整合钉钉"悟空"与阿里云"MuleRun"，冲击"OS 级"企业 Agent。
- **详情**：阿里巴巴 7 月 8 日确认，将旗下三款企业级 Agent 产品——**QoderWork**（2026 年 1 月上线，阿里云 Qoder 系列桌面 AI 智能体）、**悟空**（2026 年 3 月上线，钉钉孵化的企业协同 Agent）、**MuleRun**（2025 年 9 月上线，Agent 执行引擎）——正式合并，由新任钉钉 CEO **陈宇森**（92 年生）全面统管。整合方案：以 QoderWork 为基础底座，融合"悟空"的企业协同能力和"MuleRun"的 Agent 执行引擎能力，打造面向企业生产力场景的"OS 级"桌面智能体。同日**阿里巴巴-W 港股暴涨 12.2%**，恒生科技指数大涨 4.97%。
- **影响**：这是钉钉换帅后阿里 AI to B 战略从"多点试探"转向"重点突破"的关键动作，将直接对标 Salesforce Agentforce 与微软 Copilot。三个团队原分属不同 BU，本次整合意味着阿里下决心把 Agent 产品线收拢到单一 P&L。

### 3. 微软自研 MAI 模型开始接手 Excel/Outlook，逐步替换 OpenAI 与 Anthropic
- **要点**：每周已有数万条 AI 提示由 MAI 处理，微软"去 OpenAI 化"进入生产环境阶段。
- **详情**：据彭博社 7 月 7-8 日报道，微软已开始在 **Excel** 和 **Outlook** 等亿级用户量的核心 Office 产品中，逐步用内部自研的 **MAI（Microsoft Advanced Intelligence）系列模型** 替换此前依赖的 OpenAI 与 Anthropic 第三方模型。知情人士透露当前每周已有**数万条 AI 提示词请求**完全由 MAI 完成处理——此规模此前从未公开披露。该项目由微软 AI 超级智能团队负责人 **Mustafa Suleyman** 主导，目标是"减少并最终消除"外部 AI 供应商依赖，年降本估算数亿美元。此前微软已在 4 月发布 MAI-Transcribe-1、MAI-Voice-1、MAI-Image-2 三款自研基础模型。
- **影响**：微软是 OpenAI 最大商业客户与股东，此举加剧全球最大 AI 商业同盟裂痕；同时给企业客户传递"平台无锁定"的重要信号，也进一步压制 OpenAI 未来估值预期。

### 4. Norm AI 完成 1.2 亿美元 C 轮，法律 Agent 独角兽估值 12 亿美元
- **要点**：Khosla Ventures 领投，Full-Stack 法律 AI 模式获顶级 VC 加码。
- **详情**：位于纽约的法律与合规 AI 公司 **Norm AI** 7 月 7 日宣布完成 **1.2 亿美元 C 轮融资**，投后估值 **12 亿美元**。本轮由 **Khosla Ventures** 领投，Bain Capital Ventures、Coatue、Vanguard 参投。Norm 主打把美国 SEC、FINRA 等监管法规转化成可执行的**合规 Agent**，覆盖投行营销素材审查、员工邮件监控、保险理赔文档审计等场景，并配套一家 AI-native 律师事务所。此轮资金将用于扩员、拓宽法律实践领域、并深化在受监管行业的 Agent 部署。
- **影响**：法律与合规是 2026 年 Agent 商业化最成熟的垂直之一。Norm 的"平台 + 律所"Full-Stack 模式对通用法律 AI（如 Harvey，估值 50 亿美元）提出差异化挑战，反映资本正押注**垂直深耕型 Agent**。
