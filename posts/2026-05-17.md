### 2. 开源框架 LangGraph 2.0 发布
- **要点**：原生支持持久化、断点恢复、多分支协同。
- **详情**：新增 Checkpointer 持久化、Time-Travel 调试与 Subgraph 嵌套，配套 LangSmith 全链路 trace。
- **影响**：开源 Agent 工程化基础设施日趋完整，大幅降低复杂工作流开发成本。
- 🔗 [LangGraph 2.0 Release Notes](https://github.com/langchain-ai/langgraph)

### 3. CrewAI 完成 4500 万美元 B 轮融资
- **要点**：多 Agent 协作框架明星项目获红杉领投。
- **详情**：本轮估值 4.5 亿美元，资金用于 SaaS 产品化、企业版 SLA 与 EU 区合规。
- **影响**：多 Agent 编排赛道首个独角兽预备役，预计将带动同类项目（AutoGen、MetaGPT）二级估值。
