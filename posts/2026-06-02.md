# 🤖 今日 AI 要闻 · 2026-06-02

> MiniMax 开源新一代旗舰 M3 模型剑指 GPT-5.5；Cognition 以 260 亿美元估值完成 10 亿美元融资，Devin 89% 自我代码增长引爆 Coding Agent 赛道。

## 🎯 Agent 重点

### 1. MiniMax 发布开源旗舰模型 M3，主打编程与 Agent 能力
- **要点**：国产首个集齐前沿编程、1M 上下文与原生多模态的开源模型，瞄准 Coding Agent 场景。
- **详情**：稀宇科技（MiniMax）于 6 月 1 日正式发布新一代通用大模型 **MiniMax M3**，采用自研稀疏注意力架构 MSA（MiniMax Sparse Attention），在 1M token 上下文下预填充加速 9.7 倍、解码加速 15.6 倍。在真实软件工程基准 **SWE-Bench Pro 上得分 59.0%**，小幅超越 GPT-5.5 与 Gemini 3.1 Pro，逼近 Claude Opus 4.7。同时 12 小时无人干预自动炼出 4 个候选模型，CUDA 内核加速达 9.4 倍。
- **影响**：MiniMax 在科创板 IPO 辅导启动后立刻甩出技术王炸，把"长上下文+Agent+多模态"三件套首次合体开源，国产开源阵营格局或被重塑。
- 🔗 [搜狐：MiniMax M3 发布编程超 GPT-5.5，开源 1M 上下文](https://www.sohu.com/a/1030613843_122066678)
- 🔗 [腾讯新闻：MiniMax M3 来了，CUDA 内核加速到 9.4 倍](https://news.qq.com/rain/a/20260601A05BUS00)

### 2. Cognition 完成 10 亿美元融资，Devin 估值飙至 260 亿美元
- **要点**：Devin 母公司单笔融资 10 亿美元，估值 18 个月内翻 13 倍，Coding Agent 赛道头部位次彻底改写。
- **详情**：Cognition Labs 于 5 月 27 日宣布完成新一轮融资，**投前估值 250 亿美元、投后 260 亿美元**，过去 12 个月营收从 3700 万美元飙升至 **4.92 亿美元**，估值/收入倍数高达 53x。公司披露 **Devin 当前已自动撰写自身 89% 的代码**，并将与 Windsurf IDE 深度整合。Goldman Sachs 等顶级机构领投。
- **影响**：自主 Coding Agent 路线（Devin）与 IDE 增强路线（Cursor）形成 26B vs 99B 双王格局，资本对"AI 替人写代码"叙事完成定价；下一阶段比的是营收转化与企业落地。
- 🔗 [TechCrunch：Cognition raises $1B at $26B valuation](https://techcrunch.com/2026/05/27/ai-coding-startup-cognition-raises-1b-at-25b-pre-money-valuation/)
- 🔗 [The Next Web：Devin 写了 90% 自己的代码](https://thenextweb.com/news/cognition-just-raised-1-billion-at-a-26-billion-valuation-and-90-of-its-own-code-is-written-by-its-ai)

### 3. Anthropic 发布 Claude Opus 4.8，Coding 与 Agent 能力再升级
- **要点**：Opus 4.8 在编程、推理与 Agent 工作流方面全面增强，支持数百并行子代理。
- **详情**：Anthropic 于 5 月 28 日（北京时间 5 月 29 日凌晨）发布 **Claude Opus 4.8**。新版在 Claude Code 中引入研究预览版的"动态工作流"功能，**单任务可调度数百个并行 sub-agent**；模型诚实度提升 4 倍、价格保持不变。已通过 claude.ai、Claude Code、API、Bedrock、Vertex AI 全平台开放。
- **影响**：与同周完成的 650 亿美元融资协同发力，Anthropic 把"长时间运行 + 大规模并行 Agent"作为下一个战略锚点，与 OpenAI Operator、Google A2A 形成三足鼎立。
- 🔗 [Anthropic 官方：Introducing Claude Opus 4.8](https://www.anthropic.com/news/claude-opus-4-8)
- 🔗 [开源中国：诚实度提升 4 倍，动态工作流](https://www.oschina.net/news/447777/claude-opus-4-8)
