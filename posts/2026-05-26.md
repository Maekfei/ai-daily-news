## 🎯 Agent 重点

### 1. Google DeepMind 公开"大规模 Agent 系统"工程实践
- **要点**：DeepMind 在 AI Engineer 大会首次完整披露生产级多 Agent 系统的架构与运维经验。
- **详情**：分享内容覆盖**分布式调度、容错回放、资源池化和长任务记忆**四大模块，并提出 "Heterogeneous Intelligence（异构智能）" 概念，将其视为继 test-time scaling 之后的下一种推理扩展范式——用不同规模/模态/专业度的子 Agent 组合，替代单一大模型重复推理。配合本周早些时候发布的 Gemini 3.5 Flash + Antigravity 2.0 工具栈，Google 正把"Agent 时代的开发者基础设施"作为差异化卖点。
- **影响**：把过去靠经验摸索的多 Agent 工程问题（路由、隔离、回放、降级）首次系统化，为 LangGraph / CrewAI / AutoGen 这类框架提供了可参考的工业蓝本。
- 🔗 [TechCrunch：With Gemini 3.5 Flash, Google bets next AI wave on agents](https://techcrunch.com/2026/05/19/with-gemini-3-5-flash-google-bets-its-next-ai-wave-on-agents-not-chatbots/)

### 2. Anthropic 完成 3 亿美元收购 Stainless，收紧 SDK 与 MCP 控制权
- **要点**：Anthropic 把官方 SDK 和 MCP 服务器背后的核心创业公司收编，**对外托管服务将逐步关停**。
- **详情**：UC Today 与 Nerova.ai 报道，**3 亿美元**全现金交易上周（5 月 18-19 日）落地，Stainless 此前为多家头部 AI 公司自动生成 SDK 与 MCP 适配器。收购后 Stainless 团队全员加入 Anthropic，专注 Claude 生态底层工具链；竞争对手 OpenAI、Google、xAI 等将无法再使用其托管服务。叠加 Anthropic 6 月 15 日生效的 Claude Agent SDK / `claude -p` / GitHub Actions 计费拆分（Pro/Max 订阅不再补贴 programmatic 调用），Anthropic 正在把"Agent 时代的开发者关系"做成自己的护城河。
- **影响**：MCP 协议的"中立公共基础设施"叙事被打破——Anthropic 既是协议主导者，也将成为最大托管方，后续生态分化加剧。
- 🔗 [UC Today：Anthropic Acquires Stainless for \$300M](https://www.uctoday.com/productivity-automation/anthropic-acquires-stainless-the-sdk-and-mcp-startup-behind-every-official-claude-api-library/)

### 3. 月之暗面 Kimi 完成 20 亿美元新融资，估值 200 亿、冲刺港股 IPO
- **要点**：国资入局，半年累计融 39 亿美元，Kimi 重心从聊天问答转向**长链路 Agent 场景**。
- **详情**：腾讯网、IT之家与 36氪报道，月之暗面新一轮 20 亿美元融资进入收尾，**投后估值约 200 亿美元（约 1367 亿元人民币），半年估值翻 4 倍**；股东名单新加入国智投、北京人工智能基金、中国移动等国资央企。同步开源 Kimi K2.6，并启动港股 IPO 架构重组（红筹拆解，与中金、高盛接触）。公司明确把 K2.5/K2.6 的工具调用与长 horizon 任务能力作为下一阶段商业化主线。
- **影响**：国资资金与红筹重组叠加，意味着 Kimi 正从"创业公司"切换为"国家队选手"，对国内大模型上市路径具有标杆意义。

### 4. 阿里 Qwen3.7-Max 登顶 Arena 国产榜，主打 35 小时长 horizon Agent
- **要点**：在阿里云峰会"Agentic Cloud"主题下发布，**支持单任务 35 小时、1000 次工具调用**。
- **详情**：界面新闻报道，Qwen3.7-Max 在 Arena 全球大模型盲测总榜超过 Kimi-K2.6、DeepSeek-V4-Pro、GLM-5.1，位列国产模型第一。配套发布的 Agentic Cloud 包含 32 项新品，涵盖运行时（Agent Runtime）、编排（Multi-Agent Orchestrator）、记忆、安全、数据平面六大能力，并搭配自研 AI 芯片"真武 M890"与"磐久 AL128"超节点。35 小时 / 1000 次工具调用是关键质变信号——Agent 从"答题"走向"独立完成工程"。
- **影响**：阿里把"芯片-云-模型-Agent"做成全栈闭环，对腾讯云、字节火山、华为云形成直接压力；MCP 之外的私有 Agent 协议也开始出现。
