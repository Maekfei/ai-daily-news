## 🎯 Agent 重点

### 1. Andrej Karpathy 正式加盟 Anthropic，强化前沿 LLM 研究阵容
- **要点**：前 OpenAI 创始成员、特斯拉 Autopilot 前负责人 **Andrej Karpathy** 官宣加入 **Anthropic**，时点恰在 Google I/O 2026 开幕前几个小时。
- **详情**：5 月 19 日 Karpathy 在 X 平台发布加入 Anthropic 的声明，称"未来几年 LLM 前沿研究尤其具有塑造性"。这是继 OpenAI 联合创始人 John Schulman、Jan Leike 之后，又一位 OpenAI 系核心研究者流向 Anthropic，也是 Anthropic 在面向 IPO 的窗口期对**研究品牌**的关键加固。Anthropic 当前估值传出 **9000–9500 亿美元**区间，公司近期已把 Claude Code 作为 Agent harness 旗舰产品持续投入。
- **影响**：Anthropic 在"研究文化"上的吸引力进一步压过 OpenAI，对 Claude 模型在推理 / 代码 / 长 horizon Agent 三条主线的迭代节奏将有直接拉动；同时也让 Google 在 I/O 上推出 Antigravity 2.0 反击 Claude Code 的动机变得更强。
- 🔗 [36 氪 - 谷歌打响反击战，Antigravity 2.0 上手：Karpathy 加入 Anthropic 是步聪明棋](https://www.36kr.com/p/3817576331747331)

### 2. Google Antigravity 2.0 把 Agent 拉出 IDE，独立成跨平台桌面工作台
- **要点**：Google 在 I/O 2026 上发布 **Antigravity 2.0**，从 IDE 插件升级为**独立桌面应用 + CLI + 本地可部署 SDK**，对标 Claude Code / OpenAI Codex。
- **详情**：5 月 20 日 Google I/O 2026 开发者大会上，Antigravity 2.0 桌面版同步上线 macOS / Windows / Linux，核心定位从"面向开发者的 Agentic IDE"转向"通用 Agent-first 工作平台"：用"项目"概念取代"工作区"，单项目可跨多个本地文件夹并对每个文件夹独立配置读写权限；主 Agent 可动态调度多个**子 Agent 并行**执行；同步开放 **Antigravity CLI** 与可接入 Gemini API 的 **Managed Agents**。Demo 现场用 **93 个 Agent、2.6 亿 token、不到 1000 美元**算力，在 12 小时内从零构建了一个可运行的简化操作系统。
- **影响**：Google 把"Agent Harness"从 Cursor / Claude Code 占据的 IDE 内嵌形态，拉到与编辑器解耦的桌面级工作台，重新定义"AI 原生开发环境"的形态边界，对 Cursor、Devin、Factory 等独立 Agent 公司构成正面竞争。
- 🔗 [新浪科技 / IT 之家 - 谷歌发布 Antigravity 2.0：优化多智能体编程、新增定时任务](https://finance.sina.com.cn/tech/digi/2026-05-20/doc-inhynmhv9676595.shtml)
- 🔗 [腾讯新闻 - 谷歌 Antigravity 2.0 拆掉 IDE 变身独立工作台](https://news.qq.com/rain/a/20260520A02HU100)

### 3. 高德地图发布 amap-lbs-skill：把 LBS 数据全面开放给中文 AI Agent
- **要点**：高德地图正式上架 **amap-lbs-skill 2.0.0**，让中文 AI Agent 通过统一 Skill 协议直接调用 POI 搜索、路径规划、热力图等 LBS Web 服务。
- **详情**：amap-lbs-skill 通过 `clawhub install amap-lbs-skill` 一行命令接入，覆盖 POI 搜索（支持关键词、城市限定、类型筛选）、周边搜索（基于坐标 + 半径）、四种模式路径规划（步行/驾车/骑行/公交）、智能旅游规划、热力图数据可视化，并支持配置本地持久化。这是高德 5 月 15 日"空间智能开放平台发布会"上 **MCP + A2A 协议** 标准化连接战略的延续——高德此前已上架 amap-jsapi-skill（前端地图开发）等多款官方 Skill，形成 AMAP 与企业 AI 的双向协作体系。
- **影响**：在国内大厂中，高德是首批以"Skill / MCP 包"形态把生活服务核心数据接入 Agent 生态的玩家，给本地生活、出行、文旅类 AI 应用提供了即插即用的真实数据底座，也为中文 Agent 生态补齐了"位置"这一长期缺失的能力维度。
- 🔗 [网易 - 高德发布空间智能平台，MCP 与 A2A 协议构建 AMAP 与企业 AI 双向协作](https://www.163.com/dy/article/KT02PMUU053194QH.html)
