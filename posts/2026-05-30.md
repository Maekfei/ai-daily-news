## 🎯 Agent 重点

### 1. Anthropic 完成 650 亿美元 H 轮，估值反超 OpenAI 登顶 AI
- **要点**：投后估值 **9650 亿美元**，超过 OpenAI 的 8520 亿美元，年化营收同步冲到约 **400 亿美元**。
- **详情**：成立仅五年的 Anthropic（Claude 母公司）正式完成 H 轮 **650 亿美元**融资，投后估值较 2 月份 3800 亿美元增长 **154%**，首次成为全球最具价值的私营 AI 公司。融资同步披露其年化收入从 2025 年 12 月的约 90 亿美元飙升至 2026 年 5 月的约 **400 亿美元**，主要由 Claude Code、企业 API 与 Claude for Financial Services 等 Agent 产品贡献。公司此前已与 Google（追加最高 400 亿美元 TPU 投资）、Amazon（Trainium 训练）形成多硬件训练栈。
- **影响**：标志着 Agent / 编程类产品（Claude Code 6 个月年化收入超 10 亿美元）已成为 AI 商业化最快的赛道；Anthropic 与 OpenAI 的"双寡头"格局进入估值与 IPO 角力新阶段。

- 🔗 [36 氪 · Anthropic 链上 Pre-IPO 估值飙升至 1.2 万亿](https://www.36kr.com/p/3799097984080899)

### 2. 阶跃星辰开源 Step 3.7 Flash，瞄准生产级 Agent 工作流
- **要点**：稀疏 MoE 架构，总参数 **196B + 1.8B（ViT）**，激活仅 **11B**，最高 **400 tokens/s**，Apache 2.0 协议开源。
- **详情**：阶跃星辰（StepFun）正式发布并开源 Step 3.7 Flash 多模态模型，定位"面向生产级 Agent 工作流的 Flash 级能力"。模型采用稀疏 MoE 架构，11B 激活参数实现接近旗舰的推理质量，最高生成速度 **400 Tokens/s**，重点优化工具调用、长任务编排与视觉推理；同时开源训练 / 推理工具链。这是阶跃在阿里 Qwen3.5、智谱 GLM-5、月之暗面 Kimi K2.6、DeepSeek V4 之后的又一开源 Flash 级模型。
- **影响**：把"开源 + 高吞吐 + Agent 友好"组合再下沉一档，进一步压低国内厂商部署生产级 Agent 的算力门槛；与今天阿里 Qoder Cloud Agents、腾讯 Hy-Memory 形成"模型 + Agent 平台 + 记忆插件"的完整国产栈。
- 🔗 [阶跃星辰 · Step 3.7 Flash 模型主页](https://platform.stepfun.com/docs/llm/text)

### 3. 阿里 Qoder 上线 Cloud Agents，企业 Agent 上线周期从 1 月压到 1 天
- **要点**：阿里云 Qoder 推出全托管 AI Agent 运行平台，提供 Agent 底座 + 模型服务 + 运行环境，**API 直调**。
- **详情**：5 月 28 日，阿里云旗下编程平台 Qoder 正式发布 **Cloud Agents**，企业可通过标准化 API 直接调用全托管 Agent 服务，把传统 Agent 从开发到上线的周期从约 **1 个月**缩短到 **1 天**。Cloud Agents 与同期发布的海外 AI 产品官网 **Qwen Cloud**、通用桌面智能体 **QoderWork**、Agent 产品 **MuleRun** 形成阿里云 Agent 全家桶；面向 Agent 的云基础设施同步升级，瞄准千行百业 Agent 落地场景。
- **影响**："Agent 即服务"开始走向标准化产品形态，国内云厂商与微软 Agent 365、AWS Bedrock Agents 直接对位竞争；中小企业的 Agent 集成门槛大幅下降。
- 🔗 [新浪财经 · 阿里 Qoder 推出 Cloud Agents Agent 上线仅需 1 天](https://finance.sina.com.cn/tech/digi/2026-05-28/doc-inhzmtwm1087432.shtml)
- 🔗 [ITBear · Qoder Cloud Agents 上线助力企业 Agent 开发周期大幅缩短](https://www.itbear.com.cn/html/2026-05/1364429.html)

### 4. 腾讯混元发布 Hy-Memory，长期协作型 Agent 的"第二大脑"
- **要点**：专为 OpenClaw 这类长期协作 Agent 设计的记忆插件，**Token 消耗降低 35%、记忆更新提速 20%、记忆密度提升 45%**。
- **详情**：腾讯混元于 5 月 28 日正式推出 **Hy-Memory** 智能记忆插件，针对长期协作 Agent 普遍存在的"三周轨迹"问题（第一周惊艳、第二周遗忘漂移、第三周被降级）。框架引入"演化链 × 双系统认知"架构，在超长上下文场景下，相比基线方案 **Token 消耗下降 35%、记忆更新提速 20%、记忆密度提升 45%**。Hy-Memory 已作为长期记忆插件接入腾讯自研 Agent 框架 **Openclaw**，并对外开放 API。
- **影响**：在 Anthropic 推 Dynamic Workflows、Google 推 Gemini Spark 之际，国产侧把 Agent 工程的瓶颈从"上下文窗口"转向"工程化记忆系统"，与昨日发布的 Claude Opus 4.8 形成对照路径。
- 🔗 [新浪财经 · 腾讯混元正式推出 Hy-Memory](https://finance.sina.com.cn/roll/2026-05-28/doc-inhznkuh8471070.shtml)
- 🔗 [ChinaZ · 记忆密度升 45%、Token 省 35% 的 Hy-Memory](https://www.chinaz.com/ainews/28443.shtml)
