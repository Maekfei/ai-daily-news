## 🎯 Agent 重点

### 1. Anthropic 发布 Mythos 级模型 Claude Fable 5 / Mythos 5
- **要点**：Anthropic 首次将 Mythos 级模型大规模开放给公众，主打长任务自主执行与软件工程能力。
- **详情**：北京时间 6 月 10 日凌晨，Anthropic（Claude 母公司）同时上线 **Claude Fable 5** 与 **Claude Mythos 5**：前者面向所有 Claude 付费用户开放，后者通过 Project Glasswing 项目继续向少量网络安全防御方与基础研究合作伙伴受限发布，二者共享底层模型。官方声称 Fable 5 在软件工程、知识工作、视觉、科学研究等几乎所有主流基准上领先此前 Opus 4.8，**自主运行时长**显著拉长，token 效率优于以往 Claude。价格方面，每百万输入/输出 token 定价 **10/50 美元**，相较 Opus 4.8（5/25 美元）翻倍。
- **影响**：Mythos 级公开化标志 Anthropic 把"危险能力 + 安全护栏"的工程套路跑通，给 OpenAI GPT-5.5 与 Google Gemini 3.5 留下直接对标压力，也将再次推高前沿模型 API 的价格天花板。

- 🔗 [CNBC：Anthropic releases Mythos-like AI model](https://www.cnbc.com/2026/06/09/anthropic-mythos-claude-fable-5.html)

### 2. 恒生电子发布 LIGHT 6.0 AI 生态架构，启动 AI 金融科技公司转型
- **要点**：国内金融 IT 龙头宣布从"卖软件"全面转向"卖智能体"，覆盖证券、资管、投顾全链路。
- **详情**：6 月 5 日 2026 证券行业合作伙伴峰会上，恒生电子（600570）总裁范径武发布**LIGHT 6.0 AI 生态架构**，主打"全链路智能体系统"，覆盖交易、风控、投研、合规与运营场景，并宣布公司战略升级为"AI 金融科技公司"。本届峰会由财达证券协办，吸引近 **150 家券商与超 800 位行业专家**参会。同期，腾讯云亦落地"产业级金融智能体解决方案"，国产大模型调用成本进入快速下行通道。
- **影响**：金融机构 AI 改造从"概念验证"切到"系统替换"阶段——恒生此次绑定的是核心交易/资管系统，对传统金融软件商形成显著挤压，A 股金融软件板块当周出现明显结构性涨跌。
- 🔗 [经济观察网：恒生电子发布 LIGHT 6.0 AI 生态架构](http://www.eeo.com.cn/2026/0606/905119.shtml)

### 3. Microsoft Build 2026 公布 Agent Framework 多智能体新进展
- **要点**：微软在 Build 2026（6 月 2-3 日）将 Agent Framework 推向 GA，强调多智能体编排、可观测性与企业治理。
- **详情**：Microsoft Agent Framework 团队在 Build 2026 公布多智能体系统、Agent Harness 设计模式、Evals 与 Observability、开源治理等多场专题，Python/.NET 双语言支持，并配套 Microsoft Foundry 的 agent 部署管线。GitHub 仓库（**microsoft/agent-framework**）持续高频发布，已成为继 LangGraph、AutoGen 之后又一企业级开源 Agent 编排栈。
- **影响**：微软把 Agent Framework 与 Foundry / Copilot Studio 串成一条"开发-评测-部署-治理"完整链路，瞄准企业级多 Agent 落地的合规与可运维痛点，是对 OpenAI Codex Cloud / Anthropic Claude Code 工具链的差异化反击。
- 🔗 [Microsoft Agent Framework at BUILD 2026](https://devblogs.microsoft.com/agent-framework/microsoft-agent-framework-at-build-2026/)
